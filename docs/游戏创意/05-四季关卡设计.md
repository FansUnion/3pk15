# 05 · 四季关卡设计

## 铁律

- **章节之间** AI 升级
- **同一章节内部** AI 完全固定、不升级
- 章内难度微调只靠 **岩石数量缓慢变化**
- 四大章节按顺序解锁：须通关上一章全部关卡，才能开下一章
- 已解锁关卡可重复游玩

## 章节一览

### 春日（新手教学）

| 项 | 规格 |
|----|------|
| AI | 简易（随机走，不主动合围） |
| 岩石 | 0→1→2 |
| 目标狼胜率 | 65%–75% |
| 单局时长 | 约 1.2–3 分钟 |
| 作用 | 入门、熟悉规则、低压力留存 |

### 夏日（均衡博弈）

| 项 | 规格 |
|----|------|
| AI | 标准（抱团、铺防线、封堵缺口） |
| 岩石 | 2–4 分散 |
| 目标狼胜率 | 45%–55% |
| 单局时长 | 约 2.8–5.5 分钟 |
| 作用 | 持续拉扯；广告核心关卡 |

### 秋日（爽感缓冲）

| 项 | 规格 |
|----|------|
| AI | **与夏日相同的标准 AI（不升级）** |
| 岩石 | 6–8 密集切割 |
| 目标狼胜率 | 70%–80% |
| 单局时长 | 约 1.8–3.5 分钟（刷碎片最快） |
| 设计目的 | 夏日高压后，不提 AI，只用地形制造连吃通道；情绪爽关。整体解锁仍递增，不视为难度倒退 |

### 冬日（终极挑战）

| 项 | 规格 |
|----|------|
| AI | 大师（合围、预判走位） |
| 岩石 | 0 |
| 目标狼胜率 | 35%–40% |
| 单局时长 | 约 5–10 分钟 |
| 作用 | 硬核挑战；高卡关率服务广告停留 |

## 全局节奏

- 网页碎片化对局目标：**约 2–4 分钟**为主观感（冬日可更长）
- 通关发放基础碎片；回刷春日低难度快局是攒碎片核心循环（见 07）

## 源稿中的二期关卡向（本期编号文档仅登记，不展开为必做）

- 每日挑战限定关
- 按剩余羊数星级评分  

以上仍以源稿《游戏开发计划》二期列表为准；未并入本期必做规格。
