# 中国民间游戏 · 改造候选总览

> 目的：快速建立品类认识，寻找可对标「三打十五 / Fangrush」路径、适合改造并全球发行的民间机制。  
> 规则写「民间常见基线」，地区变体多，本文不作考据定论。不写工程与上架任务。

## 阅读顺序

1. 本文：筛选标准、总表、已在研、符号约定

## 备胎 / 不建议当旗舰

| 品类 | 态度 | 原因 |
|---|---|---|
| 五子棋 / 六子棋 | 不建议旗舰 | 红海，AI 期望极高 |
| 象棋 / 围棋 | 不建议 | 体量与品牌门槛过大 |
| 飞行棋 / 跳棋 | 不建议 | Ludo / Chinese Checkers 已占满 |
| 麻将 / 斗地主 / 掼蛋等 | 另开业务线 | 中国市场大，全球牌桌与合规另一套 |
| 军棋 | 后置 | 暗信息 + 联机成本高 |
| 踢毽、跳房、滚铁环等 | 不当棋类旗舰 | 肢体游戏，数字化易失味 |

合集备胎：大炮轰小兵、西瓜棋、成三棋、老虎吃羊、憋死牛——适合「乡土棋馆」合集，但全球爆款仍宜 **先做一款旗舰**。
