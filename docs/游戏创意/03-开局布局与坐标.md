# 03 · 开局布局与坐标

## 坐标系

- 行号 1–6、列号 1–6；点写作 `(行, 列)`
- 横向 = 列（X），纵向 = 行（Y）
- 全局唯一基准，棋子与岩石均按此坐标，无偏移

### 区域划分

| 区域 | 行 | 说明 |
|------|----|------|
| 羊群区 | 1–3 | 棋盘上半，AI 防守区 |
| 中场 | 4 | 过渡博弈区 |
| 狼方区 | 5–6 | 棋盘下半，玩家出生与活动区 |
