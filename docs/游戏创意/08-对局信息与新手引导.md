# 08 · 对局信息与新手引导

## 页面信息架构（设计层级，不写组件）

自上而下：

1. **顶部**：关卡 / 四季主题标题；对局中置顶 **已吃羊数、剩余羊数、关卡进度**  
2. **中部**：居中棋盘（页面主体）  
3. **底部**：碎片、皮肤、任务、广告相关入口区  

首页需有：章节选择、皮肤图鉴、每日任务等入口。

## 棋盘展示与交互

- 棋盘 **正方形等比例**，适配 PC / 手机网页，不拉伸  
- 遮挡优先级（定稿）：**岩石 > 棋子 > 棋盘线条**  
- 仅 **交点** 可点选操作；线条与「格心空白」无操作  
- 玩家 **只能选中狼**；羊、岩石不可点选操作  
- 选中狼后：合法走位高亮；可隔空吃时中间空与羊位/路径高亮提示

## 终局表现（判定见 02）

- 胜利：已吃羊 ≥ 8 → 立即结束，弹出通关与碎片奖励  
- 失败：三狼皆无合法步 → 立即失败结算  
- 无和棋界面  

## 其它对局操作

- 单局重置  
- 退出对局 → 返回关卡列表  

## 新手引导

- **春日第一关**弹出引导  
- 讲解：移动、吃子（隔空吃：狼—空—羊）基础规则  

## 设置（一期设计范围）

- 音量开关  
- 游戏操作说明弹窗  

## 源稿二期（非本期必做规格）

- 棋子/连吃/通关动画与特效  
- 四季棋盘主题纹理切换（渲染向）  
- 对局步数/时长统计面板、总对局与皮肤收集统计  
- 分享对局截图、多语言切换  
- 操作撤销（消耗广告解锁）
