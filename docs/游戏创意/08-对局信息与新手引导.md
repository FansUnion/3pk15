# 08 · 对局信息与新手引导

> 本文只描述**已落地**的信息架构与交互。未做项见 [`docs/任务/任务清单.md`](../任务/任务清单.md)。

## 页面信息架构

自上而下：

1. **顶部**：关卡 / 四季主题标题；对局中置顶 **已吃羊数、剩余羊数、回合提示**（含连吃 k/5）
2. **中部**：居中棋盘（页面主体）
3. **底部 / 壳层入口**：重置、静音、退出；站内另有碎片、皮肤、任务、设置等入口

首页有：开玩 CTA、章节/关卡、皮肤图鉴、设置等入口。

## 棋盘展示与交互（现行）

- 棋盘 **正方形等比例**，适配 PC / 手机网页，不拉伸
- 遮挡优先级（设计规格）：**岩石 > 棋子 > 棋盘线条**（实现缺口见任务 `P-A03`）
- 仅 **交点** 可点选；玩家 **只能选中狼**
- 选中狼后：合法走位高亮；可隔空吃时中间空与羊位/路径高亮
- 有走子 / 隔空吃 / 连吃反馈动画与音效；四季与皮肤主题棋盘可切换（随已解锁皮）
- **不强制有吃必吃**（规则见 [02](./02-核心对局规则.md)）：有吃时仍可选普通走一格；连吃中可结束本狼回合。连吃 UX 定稿（自动连吃设置、多分支怎么选等）见任务 `P-CHAIN-UX`

## 终局表现（判定见 02）

- 胜利：已吃羊 ≥ 8 → 立即结束，弹出通关与碎片奖励
- 失败：三狼皆无合法步 → 立即失败结算
- 无和棋界面

## 其它对局操作

- 单局重置（二次确认）
- 退出对局 → 返回关卡列表

## 新手引导

- **春日第一关**可弹出短引导（可跳过）
- 讲解：移动、隔空吃（狼—空—羊）基础规则

## 设置（现行）

- 静音开关（全局真静音）
- 玩法说明 / 快速提示入口
- 语言切换入口（Cookie / `/zh`；见 [`技术设计/06`](../技术设计/06-页面地图与SEO-GEO.md)）
