## Cover images · 封面图片

**English:** Three cover images are required, with a **consistent visual style**.

**中文：** 须提供 **3 张**封面，且**视觉风格一致**。

| Type | Aspect | Pixels |
|------|--------|--------|
| Landscape / 横版 | **16:9** | **1920×1080** |
| Portrait / 竖版 | **2:3** | **800×1200** |
| Square / 方形 | **1:1** | **800×800** |

### Don’t · 避免

**English / 中文：**

- Borders / 边框  
- Extra text outside the title (e.g. New / Updated / Play Now) / 标题外的促销字  
- Store icons / logos / 商店图标与 Logo  
- Unlicensed assets / 无版权素材  
- Blurry or pixelated art / 模糊或严重像素化  
- Plain screenshots only — prefer more creative presentation / 纯截图（建议更有创意）

### Do · 建议

**English / 中文：**

- Simple and scannable at small sizes / 简洁、小图可扫读  
- Title on the cover / 标题可上封面  
- Stylized readable fonts / 风格化且可读字体  
- Occasional cover updates OK; don’t change too frequently / 可偶发更新，勿过于频繁  
- Steam/Epic-style presentation can be a reference / 可参考 Steam/Epic 级呈现

## Preview videos · 预览视频

**English:** Provide landscape and portrait preview videos.

**中文：** 须提供横版与竖版预览视频。

| Spec | Requirement |
|------|-------------|
| Duration | **Max ~15–20 seconds** (longer clips may be cut to 20s) |
| File size | **≤ 50MB** each |
| Resolution | Landscape **1080p 16:9**；Portrait **1080p 2:3**（抓取未写死竖版具体像素，待核官网） |
| Audio | **Silent** / 无声 |
| Speed | **Do not** pre-speed-up footage — the platform may accelerate slightly / **不要**自行快进 |

### Avoid in video · 视频中避免

**English / 中文：** Opening transitions, black logo screens, letterboxing bars, default mouse cursor, “Play Now”/promo text, app/social icons.

动画视频可用，但须**代表真实玩法**且不误导。

## 与指标的关系（产品提示）

**English:** Covers drive first clicks; misleading art hurts conversion and trust. Keep covers aligned with in-game Fangrush visuals (board / wolves theme).

**中文：** 封面影响首次点击；货不对板会伤害转化与信任。与局内棋盘/三狼主题保持一致。

---

## Fangrush 提示

**English:** Produce all three stills + both silent preview videos before Full Launch materials freeze. Reuse brand art carefully so Poki square thumbs and CG three-aspect covers stay on-brand without copy-paste wrong sizes.

**中文：** Full 素材冻结前备齐三静图 + 双预览视频。与 Poki 方形缩略图共用品牌，但**尺寸与构图规格不同**，勿直接错用。
