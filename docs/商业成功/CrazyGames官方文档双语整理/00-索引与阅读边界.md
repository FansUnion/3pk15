### `distribution/crazygames/README.md` — 执行清单

**中文：** README 是 Fangrush 的**勾选清单**。本系列解释规则来源。冲突时以官网与协议为准。

### `distribution/crazygames/compliance.yaml` — 合规矩阵

**中文：** 承载必做/建议、Basic vs Full、状态与证据路径。本系列供人读；YAML 供 AI 与验收勾选。

### 与 Poki 双语整理的关系

**中文：** Poki 与 CrazyGames **各有 SDK、漏斗与商务**。通用产品质量见门户上架要点 `01`；勿把 Poki Web Exclusive / Fit 门槛套到 CrazyGames，也勿把 CG Basic Launch 当成 Poki Player Fit。
