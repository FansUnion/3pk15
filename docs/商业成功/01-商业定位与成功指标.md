# Fangrush 商业定位与成功指标

## 一、商业判断

Fangrush是一款小众但容易理解的非对称策略棋盘游戏。它的机会不在高价买断或重度运营，而在：

- 三狼对十五羊具有清楚、独特、易展示的视觉关系；
- 单局较短，适合Web和移动端即开即玩；
- 四季24关把传统规则转化为可学习的单人闯关；
- 地图、羊AI、策略教学和长期关卡可以持续扩展；
- H5门户能够提供独立开发者难以自行获得的初始流量。

主要风险同样明确：规则容易被模仿、直接付费意愿有限、独立站没有自然分发、24关和AI若缺乏挑战会被快速消耗。商业成功必须建立在“玩家愿意继续提高”上，不能靠增加广告或故意制造失败。

## 二、玩家价值

产品需要依次达到三层：

1. **规则可靠**：行动、胜负、恢复和存档一致。
2. **达到成品质量**：角色和岩石可辨、操作顺畅、声音和反馈可靠、24关有学习顺序。
3. **玩家愿意继续**：失败能学习，通关有价值，下一关、重试、皮肤和未来挑战形成自然动力。

自动化只能证明第一层和部分第二层。第三层必须通过真实玩家行为和反馈判断。

## 三、核心指标

| 环节 | 需要回答 | 代表指标 |
|---|---|---|
| 首次进入 | 玩家是否马上明白并开始 | 开玩率、首个有效移动、首次捕食 |
| 首局 | 规则和难度是否可接受 | 首局完成率、失败原因、首局时长 |
| 关卡学习 | 玩家是否在成长 | 每关尝试数、三次内通过、提示使用、难度跳失 |
| 连续游玩 | 游戏是否值得继续 | 下一关点击、失败后重试、每会话局数和时长 |
| 回访 | 内容是否有长期价值 | 次日/七日回访、继续最近关、重复游玩 |
| 收藏 | 皮肤和奖励是否有意义 | 皮肤查看、兑换、装备和目标差额理解 |
| 广告 | 收入是否没有伤害体验 | 广告机会、完成、失败、广告后继续和留存变化 |
| 平台 | 渠道是否值得继续投入 | 审核结果、曝光、点击、游玩质量和收入 |

指标必须组合解释。例如更低的玩家胜率若同时降低重试和回访，不是“难度成功”；更高广告展示若降低下一关点击，也不是商业成功。

## 四、阶段决策

- 独立站阶段验证公开体验、品牌和可玩性，不以广告收入判断成败。
- H5阶段验证陌生玩家是否愿意开始、继续和回访，以及平台广告分成是否值得维护。
- App阶段只有在H5或独立站证明确有玩家价值后启动，避免提前承担商店、原生SDK和长期维护成本。
- 若数据不达标，先修复首局、关卡、AI、反馈和素材；不依赖买量或广告频次掩盖问题。

## 五、不可牺牲的边界

- 不出售胜负优势，不通过广告提供必胜走法或削弱羊AI。
- 不让广告阻断主线、覆盖基础奖励或打断棋盘决策。
- 不暗中按付费或广告行为调整难度。
- 不为了短期收入牺牲规则公平、玩家理解和品牌信任。
- 不把平台曝光、一次模拟胜率或少量好评单独解释成产品成功。
