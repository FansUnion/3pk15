## 前置：S1 提交（动作在 00）

- 完成开发者账号与**游戏提交**，进入审核队列。  
- Portal 向导字段与 Basic 首交口径见 [CrazyGames游戏提交申请/说明与填表口径.md](../CrazyGames游戏提交申请/说明与填表口径.md)（[Submit Your Game](https://developer.crazygames.com/submit)）。
- S1 可用 S0 的公开可玩 Demo；**不等同于**已 Full 变现。  

**进入 S2a / S3 的门槛：** 游戏已提交（或已能在 Portal 上传版本）。

---

## S2a · Basic 前 / Basic 可用就绪

> Basic Launch：约 2 周有限受众测数据；**变现关闭**；**SDK 非 Basic 硬门槛**。

### 要做的事

- 准备可上传的**可玩构建**（满足 Basic 技术/质量底线：可玩、英语、稳定、非外站广告等）  
- 准备 Basic 所需的**基础说明与物料**（名称、描述等按 Portal 要求）  
- 若已提前接入广告 API：确保广告禁用时游戏仍可完整游玩（无卡关、无失效激励按钮）  
- **不必**为 Basic 强行完成 Full 级 SDK / 三封面视频（有则更好，按当时提交清单为准）  

### S2a 完成标准

**具备进入 Initial QA / Basic Launch 的可玩提交物；广告关闭时体验不坏。**

---

## S2b · Full 前必须就绪

> 获邀或准备进入 Full Launch 前完成。

### 要做的事

- 完成 **HTML5 SDK v3** 与游戏生命周期就绪（含加载与玩法起止、进度上报等验收口径）  
- 完成 **midgame / rewarded** 广告就绪；广告期静音停输入；处理无填充 / AdBlock / 冷却等失败路径  
- 遵守平台 **muteAudio** 等设置  
- 满足包体与文件数等 Technical 门槛（如初始下载、总包、文件数——以现行要求为准）  
- 保证 **AdBlock 下核心主线可完成**  
- 备齐 **三封面**（横/竖/方）与 **横+竖无声预览视频**  
- 产出干净门户构建（无 Admin / 外站广告网 / 独立站专用能力）  

### S2b 完成标准

**Full QA 所需的集成、包体、广告与素材均已就绪，可提交 Full 审核。**

---

## S3 · 审核与双阶段发布

### 要做的事

- 通过 **Initial QA**（质量/原创/英语/稳定等；过糙会被拒）  
- 进入 **Basic Launch**：监控约 2 周的平均时长、转化到游玩、留存等对标结果  
- 按结果行动：  
  - 全过 → 按邀请完成 S2b 并提交 **Full**  
  - 部分过 → 改进后再申请一轮 Basic  
  - 多数不过 → 不能进 Full；若再交须当新游戏并含重大改进  
- 完成 **Full QA**（含完整实现审查）  

### S3 完成标准

**通过 Full QA（或明确 Basic 未过线后的改进/重提路径）。**

漏斗说明见 [门户上架要点 · 03 · C1](../门户上架要点/03-CrazyGames专属事项.md)。

---

## S4 · 变现、收款与更新

### 要做的事

- 确认 **Full Launch 后变现已开启**（视频广告等按平台规则）  
- 在 Developer Portal 完成 **Billing / Tipalti** 配置（常见最低提现门槛约 €100；以现行页为准）  
- 建立更新流程：随时可上传新版本并提交审核；知悉更新通常较快上线、缓存刷新后可见  
- 用后台数据与反馈迭代内容、引导与变现节奏  
- 商务上继续遵守双门户**非独家**战略，除非书面重审后另签  

### S4 完成标准

**全球向 Full 已上线且可变现；收款通道已配置；可持续提交更新。**

## 首发可延后（非 S2b 硬门槛）

- User 登录 / 好友  
- Data 云存  
- Banner（可先做视频广告）  
- 内购（邀请制）/ 排行榜  
- 多人 / happytime 等非棋盘刚需

## 阶段对照（CrazyGames）

| 阶段 | 核心产出 |
|------|----------|
| S1（见 00） | 账号 + 游戏已提交 |
| S2a | Basic 可用可玩包与基础物料 |
| S2b | Full 级 SDK/广告/包体/三封面+双视频 |
| S3 | Initial QA → Basic → Full QA |
| S4 | 变现 + Tipalti + 可更新 |
