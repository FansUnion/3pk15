## P4D 五关测试总览

**English:** In Poki for Developers (P4D), each game progresses through five levels: (1) Add your game → (2) Feedback & Playtesting → (3) Player Fit Test → (4) Web Fit Test → (5) Final Poki Review.

**中文：** 在 Poki for Developers 中，游戏依次经历五关：(1) 添加游戏 → (2) 反馈与试玩 → (3) Player Fit 测试 → (4) Web Fit 测试 → (5) Final Poki 终审。

---

## Level 2 · Playtesting · 试玩测试

### 概述

**English:** Poki Playtesting delivers ~10 video recordings per run from real Poki players (inputs, console output, duration, country, device). First results often arrive within minutes. Use it to find bugs, test onboarding, balance difficulty, and validate core fun.

**中文：** Poki Playtesting 每次约提供 10 条真实玩家录像（含按键、控制台、时长、国家、设备）。通常数分钟内即有首批结果。用于查 Bug、测引导、调难度、验证核心乐趣。

### 如何运作

**English:** Choose target devices and audience category (e.g. platformer players). Players see a Mystery Tile; optional session recording. You review recordings in the P4D Playtests tab.

**中文：** 选择设备与受众类别（如平台类游戏玩家）。玩家看到 Mystery Tile，可选录制会话。在 P4D「Playtests」标签查看录像。

### 常见测试类型

**English:**

| Type | Goal | What to look for |
|------|------|------------------|
| **Baseline** | Early fun & fit | Session length; several minutes = positive signal |
| **Onboarding** | Tutorial clarity | Drop-off under 1 minute → fix tutorial |
| **Difficulty** | Balance | Repeated fails or boredom at same stage |
| **Bug** | Cross-device issues | Console errors in recordings |

**中文：**

| 类型 | 目的 | 关注 |
|------|------|------|
| **基准** | 早期趣味与匹配 | 游玩时长；持续数分钟为积极信号 |
| **引导** | 教程清晰度 | 不足 1 分钟流失 → 改引导 |
| **难度** | 平衡 | 同阶段反复失败或无聊 |
| **Bug** | 跨设备问题 | 录像中的控制台错误 |

**English:** Iteration is the point — analyze, change, test again. Playtests stay available at every stage.

**中文：** 核心是迭代 — 分析、修改、再测。各测试阶段均可继续申请 Playtest。

## Level 3 · Player Fit Test · 玩家匹配测试

### 是什么

**English:** Automated test with a **sample of 500 players**. Measures **playtime distribution** (no video). Usually completes in ~**5 hours** (varies by traffic, category, queue). Limit: **2 requests per day** per developer.

**中文：** 面向 **500 名样本玩家** 的自动化测试，统计**游玩时长分布**（无录像）。通常约 **5 小时**完成（视流量、类别、排队而定）。开发者目前**每日最多 2 次**请求。

### 解锁条件

**English:** Watch at least **5 playtest recordings** in Level 2 before requesting Player Fit Tests.

**中文：** 须先在 Level 2 **观看至少 5 条** Playtest 录像，方可申请 Player Fit Test。

### 如何读结果

**English:** Histogram of playtime buckets across 500 players (e.g. 1.5 min → 1–2 min bar). Compare two versions side-by-side at the top of the chart.

**中文：** 500 人的游玩时长直方图（如 1.5 分钟计入 1–2 分钟柱）。可在图表顶部并排对比两个版本。

### 进入 Web Fit 的门槛指标

**English:** To advance to **Web Fit Test**, your game must achieve:

1. **Average overall playtime > 3 minutes**
2. **At least 25% of the 500 sessions last > 3 minutes**

**中文：** 进入 **Web Fit Test** 须同时满足：

1. **平均总游玩时长 > 3 分钟**
2. **500 次游玩中至少 25% 超过 3 分钟**

**English:** Meeting these shows genuine interest, not just a small group playing excessively. Failing first attempts is normal — iterate onboarding, content, and core loop; use more Playtests before retesting.

**中文：** 达标说明有真实兴趣，而非少数玩家刷时长。首次未达标很常见 — 迭代引导、内容与核心循环；重测前可多做 Playtest。

### 如何改进

**English:** Right-side bars (especially **5m+**) = stronger performance. Most players in first two columns → fix onboarding/early game. Many in middle columns → may need more content or stronger hooks. Request Playtests to validate changes before another Player Fit run.

**中文：** 图表**右侧**（尤其 **5m+**）越高越好。多数人集中前两列 → 改引导/前期体验。中间列很多 → 可能缺内容或留存钩子。下次 Player Fit 前可用 Playtest 验证改动。

## Level 4 · Web Fit Test · 网页匹配测试

### 概述

**English:** Game is shown to ~**10,000 players** on a **small set of category pages**, appearing like a live release (not full homepage promotion). Provides unbiased fit assessment for Poki’s audience and category placement. Duration: ~**3–5 days**; **cannot stop** once started. You may upload a new build mid-test and email developersupport@poki.com to swap the live version.

**中文：** 游戏在**少量分类页**面向约 **1 万玩家**展示，外观接近正式上架（非全站首页推广）。用于无偏评估与 Poki 受众/分类匹配。约 **3–5 天**；**开始后不可中止**。测试中可上传新版本并邮件 developersupport@poki.com 请求切换。

### 解锁条件

**English:** Watch ≥ **5 playtest recordings** (Level 2), **pass Player Fit** (Level 3), and **upload a static thumbnail**.

**中文：** Level 2 观看 ≥ **5 条** Playtest、**通过 Player Fit**（Level 3），并**上传静态缩略图**。

### 测试期间变现

**English:** `commercialBreak()` / `rewardedBreak()` are **not required** during Web Fit. If implemented, **placeholder ads** show — **no revenue** is earned or shared in this stage.

**中文：** Web Fit 阶段**不强制**实现插播/激励事件。若已实现，显示**占位广告**，**不产生也不分成**收入。

### 三项同等重要指标

**English:**

| Metric | Meaning |
|--------|---------|
| **CTR** (Click-through Rate) | % of category visitors who click your thumbnail |
| **Average Time on Page** | How long players stay on your game page |
| **Conversion to Play (C2P)** | % who reach first `gameplayStart()` — reflects load speed & onboarding |

**中文：**

| 指标 | 含义 |
|------|------|
| **CTR**（点击率） | 分类页访客中点击你缩略图的比例 |
| **Average Time on Page**（页面平均停留） | 进入游戏页后的平均停留时长 |
| **C2P**（转化到游玩） | 触发首次 `gameplayStart()` 的比例 — 反映加载与引导 |

**English:** Each metric is scored **0–5** vs category averages. **Score ≥ 3** = at or above average for tested categories. Example benchmarks (Action + Adventure): CTR 1.5%, Time 4 min, C2P 70% → score 3 if matched.

**中文：** 各指标按所测分类均值评 **0–5 分**。**≥ 3 分** 表示达到或超过同类均值。示例（动作+冒险）：CTR 1.5%、停留 4 分钟、C2P 70% 即约 3 分。

### 测试结束后

**English:**

- **Pass:** Auto-submitted to **Final Poki Review** (not instant global release).
- **Fail:** Iterate using metrics; **retry Web Fit** with a new version if still within Quality Guidelines.
- Between pass and final review, game may stay live on test categories **without promotion or ad earnings** so testers can keep playing — contact Poki to remove if needed.

**中文：**

- **通过：** 自动进入 **Final Poki Review**（非立即全球发布）。
- **未通过：** 按指标迭代；若仍符合质量指南可**再次申请 Web Fit**。
- 通过至终审期间，游戏可能在测试分类保持在线但**无推广、无广告收入**；如需下架可联系 Poki。

## 未过 Web Fit 怎么办

**English:** Failure is not a verdict that your game is bad — it needs more work for **this platform**. You **must pass Web Fit** to reach Final Review.

**中文：** 未通过不代表游戏不好，而是尚需适配**该平台**。**必须通过 Web Fit** 才能进入终审。

**English:**

- **Low CTR** → Review [thumbnail best practices](https://sdk.poki.com/) (see doc 06).
- **Low Average Playtime** → Revisit Playtesting & Player Fit; improve retention/content.
- **Low C2P** → Faster loads, smaller files, shorter onboarding.
- **Keep iterating** — more Playtests and Player Fit Tests remain available.

**中文：**

- **CTR 低** → 复查缩略图规范（见 06 篇）。
- **停留时长短** → 回到 Playtest / Player Fit，加强留存与内容。
- **C2P 低** → 缩短加载、减小体积、精简引导。
- **持续迭代** — Playtest 与 Player Fit 仍可继续使用。

## Level 5 · Final Poki Review · 终审

**English:** After all testing stages, submit for release. Poki reviews **test results** and **game content** against Quality Guidelines and **uniqueness within category**. Timeline: **1–2 weeks**.

**中文：** 测试通过后提交发布申请。Poki 审核**测试结果**与**内容**（质量指南、分类内**独特性**）。周期约 **1–2 周**。

**English:**

- **Pass:** Team contacts you for **contract signing** and **release process** next steps.
- **Fail:** No publish on Poki at this time; you may submit **other games** later.

**中文：**

- **通过：** 团队联系你进行**签约**与**发布流程**后续步骤。
- **未通过：** 当前无法在 Poki 发布；可日后提交**其他作品**。

## Release Process · 发布流程

**English:** Structured, data-driven path from QA to global launch. Collaborate with Developer Support; monitor C2P, errors, monetization.

**中文：** 从 QA 到全球上线的结构化、数据驱动流程。与 Developer Support 协作，监控 C2P、错误与变现。

### 🔧 Technical Test · 技术测试

**English:** Global release limited to **New Games** category (~**5k plays/day**) to stress-test scale. Often **skipped** if prior tests suffice; may be **required** for multiplayer or novel tech. Metrics: C2P, pre-play time, errors, bugs. Request via your Poki representative.

**中文：** 全球范围但仅 **New Games** 分类（约 **5000 次/日**），验证大规模承载。多数游戏**可跳过**；多人或新技术可能**必须**。指标：C2P、预玩时间、错误、Bug。向 Poki 对接人申请。

### 🏖️ Soft Release · 软发布

**English:** Scheduled **1–2 weeks** after QA pass. Game on **relevant category pages** (not homepage/localized homepages). Duration ~**2–3 weeks**; **cannot skip**. Monitor: C2P & engagement, Error Details in P4D, monetization (**≥ 1 ad per DAU**, midroll + rewarded mix), player feedback. Upload fixes same as QA review flow. May promote to extra categories if performance is strong.

**中文：** QA 通过后约 **1–2 周**排期。上架**相关分类页**（非首页/本地化首页）。约 **2–3 周**；**不可跳过**。监控：C2P 与参与度、P4D 错误详情、变现（**≥ 1 广告/DAU**，插播+激励组合）、玩家反馈。修复流程同 QA 上传审核。表现好可能扩至更多分类。

### 🌍 Global Release · 全球发布

**English:** Full audience after soft release (~**2–3 weeks** from soft start, depending on optimization). Promotion system activates: localized homepages, categories, game pages. **~2 weeks “new game” boost** + **fire icon** on thumbnail.

**中文：** 软发布后面向全量用户（视优化进度，软发布起约 **2–3 周**）。推广系统启动：本地化首页、分类、游戏页。**约 2 周新游加权** + 缩略图**火焰标**。

### 📣 Post-Release Monitoring · 发布后监控

**English:** Keep monitoring performance, bugs, and errors. Reach out to Poki for improvement ideas.

**中文：** 持续监控表现、Bug 与错误。有优化想法可联系 Poki 团队。

## How Does Poki Promote My Game · 推广机制

### Promotion at Global Release · 全球发布期推广

**English:** Every new global release gets **2–3 weeks** of promotion: category placement, related game pages, homepage visibility (**fire icon**). Top performers vs recent releases may get **extended homepage** exposure.

**中文：** 全球发布后有 **2–3 周**推广：分类位、相关游戏页、首页曝光（**火焰标**）。相对近期新作表现突出者可能获得**更长首页**曝光。

### Post-Release Promotion · 发布后持续推广

**English:** Ongoing recommendations on category/game pages. Strong metrics (gameplays, engagement, C2P) → higher placement, homepage tiles (large/small) for category leaders. Recommender adapts weekly to daily releases; significant stat improvements are reflected within about a week. No fixed threshold for homepage — **regular content updates** and **better stats** help.

**中文：** 分类/游戏页持续推荐。指标强（游玩量、参与度、C2P）→ 更高排位、首页大/小图块。推荐系统随日更调整；指标显著提升约一周内反映。首页无固定阈值 — **定期内容更新**与**更好数据**更有利。

---

## Post-Release Updates · 发布后更新

**English:** After global release, monitor feedback and ship updates via P4D:

1. **Bugs / performance / tech / engagement / monetization** — fix quickly; better metrics unlock more promotion.
2. **New content** (levels, worlds, characters) — re-engage players; discuss strategy with Poki contact.
3. **Localization** — add languages anytime; discuss with Poki contact.

**中文：** 全球发布后通过 P4D 持续更新：

1. **Bug / 性能 / 技术 / 参与 / 变现** — 尽快修复；更好指标带来更多推广。
2. **新内容**（关卡、世界、角色）— 拉回老玩家；可与 Poki 对接人讨论策略。
3. **本地化** — 随时加语言；可与 Poki 对接人讨论是否合适。
