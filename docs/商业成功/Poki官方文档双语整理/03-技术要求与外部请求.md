**English:** First step: upload to **Poki for Developers**. Before upload, read [Terms & Conditions for Developers](https://sdk.poki.com/). This section is the non-negotiable technical, design, and policy baseline for Poki’s global player base.

**中文：** 第一步：上传至 **Poki for Developers**；上传前阅读开发者条款。本节为面向全球玩家的**硬性**技术、设计与政策基线。

## 硬性平台规则（摘要）

### Desktop, mobile and tablet support

**English:** Game must work on all web-connected devices. On mobile, cover the full screen in portrait **or** landscape—or **both** for best UX.

**中文：** 须支持桌面、手机、平板。移动端须竖屏或横屏铺满全屏，**两者都支持**体验最佳。

### Widescreen aspect ratio (16:9)

**English:** Scale to full canvas on all devices at **16:9**. Reference proportional sizes: **640×360**, **836×470**, **1031×580**.

**中文：** 全设备 **16:9** 铺满画布。参考等比尺寸：**640×360**、**836×470**、**1031×580**。

### Incognito support

**English:** Must work in incognito. Google Incognito restricts `localStorage`—wrap `localStorage` in **try/catch** (or equivalent) so play still starts.

**中文：** 须支持隐私/无痕模式。`localStorage` 可能不可用——用 **try/catch** 等处理，确保仍可开玩。

### Privacy & security (external requests)

**English:** Poki **blocks all external requests by default** from platform games. Remove unnecessary calls to Google Fonts, external assets, CDN libraries (e.g. jsDelivr), etc., before release.

**Exceptions (need up-to-date Privacy Statement for Poki):**

1. Multiplayer on externally hosted servers.  
2. Third-party analytics (e.g. GameAnalytics, in-house)—case-by-case approval.

**Google products (including Google Analytics) are blocked in all cases.**

Details: [External Resources](#external-resources) below and [official page](https://sdk.poki.com/external-resources.html).

**中文：** 平台**默认拦截一切外部请求**。发布前移除 Google Fonts、外链资源、外部 CDN 库等非必要请求。

**例外（须向 Poki 提供最新隐私声明）：** 外部服务器多人；第三方分析（个案批准）。**Google 系（含 GA）一律禁止。**

### No branding or external ads

**English:** Remove splash screens and outgoing links. Studio logo on loading screen is OK. Poki SDK includes ads—**no other ad networks** in-game.

**中文：** 移除启动外链与出站链接；加载页可放工作室 Logo。SDK 已含广告——**禁止其他广告**。

### No ad block prevention

**English:** Game must remain playable with ad blockers; **no core gameplay** gated behind ad-block messages. Poki handles this on the platform—**no custom “ad blocked” UI** (see Advertisement Integration).

**中文：** 有广告拦截插件也须能玩；**核心玩法**不得被拦截提示挡住。平台侧处理——**勿做自定义“广告被拦”提示**（见广告集成）。

### Poki Quality Guidelines

**English:** Content must meet [Quality Guidelines](https://sdk.poki.com/quality-guidelines.html) (doc 02). Developer Support checks upcoming games against guidelines + requirements below.

**中文：** 内容须符合质量指南（见 02）。Developer Support 对照下列要求审查。

## 1. Technical standards

### SDK integration

**English:** Implement SDK events correctly—**required for Web Fit and all releases**. Common mistakes:

| Rule | Detail |
|------|--------|
| No duplicate consecutive events | `gameplayStart()` cannot follow `gameplayStart()`; `gameplayStop()` cannot follow `gameplayStop()` |
| First input | `gameplayStart()` on **first player input**, not on load |
| Interruptions | `gameplayStop()` on pause, menu open, level end, cutscene, etc. |
| During ads | **No SDK events** during midrolls or rewarded videos |
| `commercialBreak()` timing | Only when **exiting pause → returning to gameplay** (e.g. close pause menu → play ✅; leave to level select ❌) |
| Edge case | “Dress-up” games may fire `gameplayStop()` + `commercialBreak()` when switching clothing categories |

Full SDK: [General Info](https://sdk.poki.com/general-info.html), [HTML5](https://sdk.poki.com/html5.html).

**中文：** SDK 事件须正确——**Web Fit 及所有发行必需**。常见错误：

- 禁止连续重复同类事件  
- `gameplayStart()` 在**首次输入**，非加载时  
- 暂停/菜单/关卡结束/过场等须 `gameplayStop()`  
- **广告播放中禁止任何 SDK 事件**  
- `commercialBreak()` 仅在**退出暂停回到玩法**时（关暂停菜单→玩 ✅；去选关 ❌）  
- 换装类游戏切换类别时可 `gameplayStop()` + `commercialBreak()`

### File size

**English:** Target initial download **under 8 MB**. (Unity has separate optimization guides.)

**中文：** 初始下载目标 **< 8 MB**。

### Clean build

**English:** Remove dev tools, debug code, and test artifacts before publication. **Fangrush:** no Admin routes, debug overlays, or standalone-site ad code in the Poki bundle.

**中文：** 发布前移除开发工具、调试代码、测试残留。**Fangrush：** Poki 包不含 Admin、调试层、独立站广告代码。

### Save system

**English:** Implement progress saving where appropriate, or clearly tell players progress won’t persist on exit.

**中文：** 适当实现存档，或明确告知退出不保存。

## 2. Platform integration

**English:**

- **Thumbnails** — Static **and** animated for global release ([Game Thumbnail Guide](https://sdk.poki.com/game-thumbnail.html)).  
- **Page integration** — Prevent game viewport scroll from scrolling the parent page during review.  
- **Mobile controls** — Force mobile control scheme on **tablets**.  
- **Responsive design** — Consistent across sizes and input methods.  
- **In-game Privacy Policy** — If any external link/analytics exemption: in-game accessible privacy UI + **hosted** policy URL (no Google Doc links).

**中文：** 须静态+动态缩略图；防止游戏内滚动带动父页面；平板强制移动操控；响应式一致；有外链/分析豁免时须游戏内隐私入口+**托管**政策 URL（禁 Google 文档链）。

## 3. User interface & experience

**English:**

- **Streamlined entry** — Minimize pre-play screens; drop players into gameplay when possible.  
- **Keyboard** — ESC or Space for pause/resume on keyboard games; fire correct SDK events.  
- **Cutscenes** — All intros/cutscenes **skippable**.  
- **Tutorials** — Visual/intuitive, not text walls.  
- **Adaptive controls** — Show mobile controls on mobile/tablet; keyboard hints on desktop.

**中文：** 减少进游戏前界面；键盘游戏 ESC/空格暂停并打 SDK 事件；过场可跳过；教程偏视觉；移动端显示触控、桌面显示键位。

## 5. Advertisement integration

**English:**

- Implement `commercialBreak()` and/or `rewardedBreak()` as appropriate.  
- **Mute game audio** during ads.  
- **No internal ad timers**—Poki controls frequency.  
- **One video per reward**—no stacking multiple rewarded videos for one reward.

**Reward UI hierarchy:**

- Always a **standard Continue** alternative to rewarded option  
- Both options visible **simultaneously**  
- Standard button **≥ size** of reward button; **green** color; **next to or above** reward button  
- Reward buttons **cannot** be green; must show prominent 🎬/🎞️ icons  
- Clear confirmation on reward (animation/sound/VFX)  
- Auto-apply rewards when possible  
- **No double-reward** for same action  
- **No reward** when ad blocker detected  
- **Silent handling**—no custom ad-block messages (Poki manages)

**HTML5 note:** Disable keyboard input and audio during `commercialBreak()` / `rewardedBreak()`; prevent Space/Arrow scroll bubbling on embedded page (see [HTML5 Final Steps](https://sdk.poki.com/html5.html)).

**中文：** 实现 `commercialBreak()` / `rewardedBreak()`；广告时静音；**禁止自建广告计时器**；一奖励一视频。

**奖励 UI：** 须有普通继续按钮（绿色、同等或更大、与奖励并列或在上）；奖励钮非绿色且带 🎬/🎞️；发放须明确反馈；尽量自动生效；禁双重奖励；检测到拦截器**不发奖励**；**静默**，勿自定义拦截提示。

**HTML5：** 广告期间禁输入与音频；嵌入页须防空格/方向键滚动父页面。

**English:**

## Contract-level performance (Deal Types 交叉)

**English:** Signed deals may additionally require: **30 FPS minimum (60 target)** on supported devices under ~3G+; **85%** of users without critical issues; bug fixes within **1 month** of report. See [Deal Types](https://sdk.poki.com/deal-types.html)—verify in contract.

**中文：** 合同可能另要求：主流设备 **30 FPS 起（目标 60）**、约 3G+ 网络；**85%** 用户无严重问题；Bug **1 个月内**修复。以 [Deal Types](https://sdk.poki.com/deal-types.html) 与合同为准。

## External Resources

**English:** To reduce privacy compliance risk, Poki **blocks all external requests by default**. Games may not call third-party URLs unless **explicitly approved**.

### Forbidden request types

**English:**

- ❌ **Unnecessary external assets** — Google Fonts, externally hosted images/audio, CDN code libraries → **bundle locally**; no runtime URL calls.  
- ❌ **In-game chat** — Risk for younger players. Not allowed except games covered by **Poki Guardian** moderation (rolling out). Prefer **emoji** expression instead.  
- ❌ **External account systems** — No collecting personal info; no email login; no Google/Facebook login; remove any identifiable PII prompts.

**中文：**

- ❌ **非必要外链资源** — 字体/图音/ CDN 库等须**打包进包体**，运行时勿请求。  
- ❌ **游戏内聊天** — 默认禁止；**Poki Guardian** 审核的少数游戏例外；可用表情符号替代。  
- ❌ **外部账号** — 禁止收集个人信息、邮箱登录、Google/Facebook 登录等。

### Exceptions

**English:**

- **Multiplayer external servers** — OK after Poki approval.  
- **Third-party analytics** — GameAnalytics, ByteBrew, in-house—case-by-case; Privacy Policy required.  
- **Leaderboards** — Username (+ profanity filter) + password, **no** personal data collection.

**Google Analytics / Google trackers: blocked.**

**中文：** 外部多人服务器、第三方分析（个案）、排行榜（用户名+密码、无 PII）可例外；须隐私政策。**Google 分析/跟踪：禁止。**

### How to get an external link approved

**English:** Request in game **Settings → Custom Content Security Policy**. Provide Privacy Policy that:

1. Hosted on a **live public webpage**  
2. **Linked inside the game**  
3. Clearly describes how the external service handles user data  

If approved, Poki configures **CSP** for your game. Poki will not review or sample your policy (conflict of interest)—you must ensure completeness.

**中文：** 在 P4D 游戏 **Settings → Custom CSP** 申请。隐私政策须：公网托管、游戏内链接、说明外部服务如何处理数据。批准后 Poki 配置 **CSP**；Poki 不提供政策范例或审阅。
