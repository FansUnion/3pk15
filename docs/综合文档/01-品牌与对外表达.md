# Fangrush 品牌与对外表达

## 一、品牌定稿

| 项目 | 当前决定 |
|---|---|
| 英文主品牌 | **Fangrush** |
| 中文展示名 | **三狼连猎** |
| 域名 | `fangrush.com` |
| 发音 | FANG-rush |
| 品类 | Strategy / Puzzle / Board game |
| 英文品牌句 | Command three wolves. Outmaneuver the flock. Chain the hunt. |
| 中文品牌句 | 操控三狼，隔空连吃破阵，闯过四季猎场。 |

“Fang”表达狼和捕食，“rush”表达直线冲向羊位以及连续捕食的推进感。品牌首先让玩家期待轻量、清楚、有节奏的策略挑战，而不是血腥狩猎或慢速传统棋谱研究。

## 二、产品表达

对外稳定事实：

- 玩家操控三只狼，对阵十五只AI羊；
- 6×6交点棋盘；
- 狼跨过一个空点落到羊的位置完成捕食；
- 捕食后可能继续连吃，也可以主动收手；
- 在三狼被全部困住前累计捕食八只羊即胜；
- 四季24关，具有固定开局、岩石布局和递进羊AI。

平台可直接粘贴的一句话、短描述和操作说明统一在 `distribution/common/materials-manifest.md`，避免品牌文档和发行目录维护两套当前文案。

## 三、术语

| 推荐 | 避免 | 原因 |
|---|---|---|
| capture / chain capture | jump over the sheep | 狼落在羊位，不是越过羊落地 |
| leap across one empty point to capture | gap-eat | 首次接触不要求玩家理解自造术语 |
| 隔空捕食 / 隔空吃 | 跳吃到羊后方 | 与真实几何一致 |
| sheep AI / flock defense | enemy algorithm | 使用玩家语言 |
| hunt / seasonal hunt | stage config | 不暴露内部实现词 |

## 四、语气

- 简洁、直接、略带狩猎张力，但不血腥。
- 先说明玩家是谁、目标是什么，再讲策略深度。
- 不宣称理论必胜、完美AI、无限关卡或不存在的多人能力。
- 中文自然表达，英文不逐字翻译内部术语。
- 平台、首页、帮助和关卡文案可以长度不同，但规则事实必须一致。

## 五、品牌边界

- `狼与绵羊`只作为内部或品类解释，不作为主品牌。
- 皮肤和季节主题可以扩展视觉世界，但不改变Fangrush名称和三狼猎阵身份。
- 新宣传文案先核对现行规则和真实功能，再进入发行目录；不要在多个产品文档建立候选文案库。
