## 2. 核心游戏规则摘要（命名必须对齐）

### 2.1 定位

- 载体：网页独立站 + Poki / CrazyGames 等 HTML5  
- 模式：纯人机闯关；玩家只操控狼  
- 一期不做：操控羊、双人、联机  
- 变现：纯广告 + 碎片皮肤（前期无付费商城）

### 2.2 棋盘与吃子

- 6×6 交点；3 狼 vs 15 羊；单点岩石  
- **隔空吃**：同线「狼—空—羊」，狼移到羊位  
- 连吃 ≤5；羊不后退；狼胜吃 ≥8；羊胜合围三狼  

### 2.3 差异化（品牌应暗示）

进攻猎手、隔空冲吃 + 连吃、岩石旋钮、四季章节、可爱外壳 + 策略猎杀。

## 3. 命名与域名硬约束

1. 只要 `.com`，品牌词与域名一致  
2. `packleap.com` 已占，Packleap 不作首选  
3. 英文主名尽量 1 词、6～12 字母  
4. 暗示优先：`fang` / `rush`·`dash` / `pack`·`tri` / `rock` / `hunt` / `season` / `grid`；**不再优先 leap**  
5. 避免护羊堵狼语义  
6. 中文展示短、动词感  
7. 独立站、Poki slug、社交柄同一词根

## 7. 评分表（满分 10）

| 维度 | 权重 | 说明 |

| 机制贴合 | 25% | 隔空冲吃 / 连吃 / 岩石，非护羊、非跳过羊 |
