## 管理规则

- 未来内容不得改变四季24关的正式规则和验收口径。
- 新模式必须使用可复现配置、固定规则入口和候选验收器。
- 决定近期开发后，在任务清单建立活动任务；完成后保留产品规格，但移除任务状态。
- 独立挑战赛的定位、禁止事项和验收边界已合并到“长期更新与无限内容架构”，不再单独维护重复方案。
