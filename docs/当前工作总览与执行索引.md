- 部署边界已确定：同一 monorepo 建两个 Vercel Project，玩家端使用 `apps/player-web`，Admin 使用 `apps/admin`；本地分别是 `5003`、`5002`。
- `fangrush.com` 玩家端和 `admin.fangrush.com` Admin 均已部署成功；后续只记录线上回归问题，不再把部署作为当前待办。
