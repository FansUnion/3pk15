# 三炮十五兵（口误名：三打十五）

双人非对称棋盘棋：**3 炮 vs 15 兵**。民间常被口误叫成「三打十五」；相近品类有海外 Fox and Geese / Bagh-Chal（多为跳吃，与本稿不同）。地区另有隔子吃、2 炮 18 兵等变体——**本项目只实现下方基线（隔空吃）**。

---

## 1. 棋盘

- 5×5 交叉点，共 25 个落子点
- 只能沿网格线走：横 / 竖，不能斜走

## 2. 开局

```
      0     1     2     3     4
 0  [　]--[炮]--[炮]--[炮]--[　]
     |     |     |     |     |
 1  [　]--[　]--[　]--[　]--[　]   ← 空行
     |     |     |     |     |
 2  [兵]--[兵]--[兵]--[兵]--[兵]
     |     |     |     |     |
 3  [兵]--[兵]--[兵]--[兵]--[兵]
     |     |     |     |     |
 4  [兵]--[兵]--[兵]--[兵]--[兵]
```

| 方 | 数量 | 位置 |
|----|------|------|
| 炮 | 3 | 第 0 行中间三点 `(0,1)(0,2)(0,3)`；两角空 |
| 兵 | 15 | 第 2～4 行占满，共 15 点 |
| 空 | 7 | 第 1 行整行 + 第 0 行两角 |

**先手：炮方。**

## 3. 行棋（双方相同，不吃子时）

- 每回合移动 **一枚** 己子
- 移到 **相邻** 空点（横或竖 1 格）
- 不吃子时 **不作** 远距离滑动

## 4. 炮吃子（隔空吃）

- 兵 **不能** 吃子，只能移动围堵
- 炮可吃兵，条件（须同时满足）：
  1. 炮与目标兵在同一横线或竖线
  2. 两者之间 **恰好有 1 个空点**（曼哈顿距离为 2，中间必须为空）
  3. 目标点是敌兵 → 炮移到该点，移除该兵
- 中间有子、中间多空、或紧贴无空：均不可吃
- **不强制**「有吃必吃」

示意：

```
合法：  炮 —— · —— 兵
非法：  炮 —— 兵              （紧贴，无空格）
非法：  炮 —— 架 —— 兵        （中间是子不是空）
非法：  炮 —— · —— · —— 兵    （中间不止一个空）
```

> 实现：同线两点距离为 2，中间点为空，目标为敌兵；吃子后炮占据该点。不吃子时仍只走相邻 1 格。

## 5. 胜负

| 结果 | 条件 |
|------|------|
| **炮胜** | 场上剩余兵 **≤ 3** |
| **兵胜** | 全部 3 枚炮均无合法行动（不能移 1 格，也不能完成一次合法吃子） |

## 6. 扩展预留（暂不实现）

规则开关可预留枚举，例如：`隔空吃`（本基线） / `隔子吃` / `跳吃`；炮胜条件 `剩余≤3` / `吃光`；子力 `3v15` / `4v30` 等。开新变种时改配置，不改默认基线。
