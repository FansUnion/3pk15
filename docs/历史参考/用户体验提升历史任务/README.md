## 保留结论

- 新玩家应快速看懂狼、羊、棋盘目标和第一步操作。
- 选中、合法行动、移动、捕食、连吃、AI回合和终局必须有清晰且不冲突的反馈。
- 手机与桌面端都要保证棋盘可操作、角色可辨认；静音和减少动效时仍能理解对局。
- 胜利、失败和外围页面都必须给出明确下一步，不能形成流程死路。
- 皮肤和奖励应支持核心玩法与回访动力，不能代替关卡质量。
