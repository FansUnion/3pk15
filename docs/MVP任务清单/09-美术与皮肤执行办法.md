# 09 · 美术与皮肤执行办法

规格：[07](../MVP技术设计/07-皮肤与资源管线.md)、界面视觉约束（牧场浅色、非紫霓虹）

## 原则

- 逻辑不等人：先用色块/几何占位通关。  
- 狼套 **必须** 同时交付羊。  
- 岩石一式到底。  
- MVP 不做远程热更。
