## 推荐并行车道

```
车道 A（逻辑）:  W0 → W1 → W2 → W5b(AI台完善)
车道 B（体验）:  W0 → W3(先假AI) → 接真AI → W4 → W5
车道 C（美术）:  占位 → default → 四季板 → frost（见 09）
车道 D（上线）:  W6（后置）
```

一人公司：按 A 主线推进，B 在 W1 可玩后切入；C 每天抽固定时间。
