# 01 · 体验标准指针（感官侧）

> 三层目标、全球对标、玩家漏斗、对局时序 → **SSOT** [`docs/商业成功/`](../商业成功/00-索引.md)（01–03）。  
> 本页只保留**感官施工不可妥协条**与测量建议，避免与商业成功重复。

## 1. 原则

1. 以普通玩家上线标准审整包，不以「规则测通」自满。  
2. 先标准与缺口，再成批施工；禁止「你指哪改哪」。  
3. 品牌/文案/视觉与玩法口径一致（Fangrush、隔空吃）；壳层禁止 MVP / 错品牌。

## 2. 七条不可妥协（本目录硬约束）

1. 棋盘是唯一主角（官网首屏与对局皆然）。  
2. 1 秒可辨狼 / 羊 / 岩（约 36px）。  
3. 对比优先于「淡雅发灰」。  
4. 一页一主动作。  
5. 反馈是事件，不是只改数字。  
6. 铬层（顶栏底栏说明）永远轻于棋盘。  
7. 跨页同气质（色板、按钮、棋子同源）。

## 3. 测量（独立站 GA4）

建议事件：`play_start`、`level_clear`、`play_again`、`skin_equip`、`locale_switch`。  
先看漏斗转化（见商业成功 02），再看美术主观分。
