# 05 · 音效资源清单与听感规格

## 1. 事件表

| 事件 | 文件 | 听感隐喻 | 时长预算 | 忌 |
|------|------|----------|----------|-----|
| step | `public/sfx/step.wav` | 轻木棋「笃」 | &lt; 120ms | 手机按键音 |
| capture | `public/sfx/capture.wav` | 短 whoosh + 轻咔 | &lt; 200ms | 枪声/血腥 |
| chain | `public/sfx/chain.wav` | 同族音高递进 | &lt; 200ms | 过长旋律 |
| win | `public/sfx/win.wav` | 短上行解决 | &lt; 500ms | 长 fanfare |
| lose | `public/sfx/lose.wav` | 短下行 | &lt; 400ms | 刺耳报警 |
| ui（可选） | `public/sfx/ui.wav` | 极轻点击 | &lt; 80ms | 抢对局声 |

## 2. 契约

- `settings.muted === true` → **完全无声**  
- 文件优先；fallback 蜂鸣仅开发兜底，**不得当成品主声**  
- 首次用户手势后再解锁 AudioContext  
- 与画面 ~200ms 反馈大致对齐  

## 3. 用户期待

戴耳机默认音量不刺耳；连吃有节奏但不烦；静音键可信。
