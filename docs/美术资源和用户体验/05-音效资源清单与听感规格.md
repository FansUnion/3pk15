# 05 · 音效资源清单与听感规格

> 代码入口：[`apps/web/src/lib/sfx.ts`](../../apps/web/src/lib/sfx.ts)。以代码事件名为准。

## 1. 事件表

| 事件（sfx.ts） | 文件 | 听感隐喻 | 时长预算 | 忌 |
|----------------|------|----------|----------|-----|
| step | `public/sfx/step.wav` | 轻木棋「笃」 | &lt; 120ms | 手机按键音 |
| jump | `public/sfx/capture.wav` | 短 whoosh + 轻咔（隔空吃） | &lt; 200ms | 枪声/血腥 |
| chain | `public/sfx/chain.wav` | 同族音高递进 | &lt; 200ms | 过长旋律 |
| win | `public/sfx/win.wav` | 短上行解决 | &lt; 500ms | 长 fanfare |
| lose | `public/sfx/lose.wav` | 短下行 | &lt; 400ms | 刺耳报警 |
| select | （无 wav） | 程序化轻音 | — | 抢对局声 |
| invalid | （无 wav） | 程序化提示 | — | 刺耳 |
| ai | （无 wav） | 程序化弱提示 | — | 像转圈 loading |

## 2. 契约

- `settings.muted === true` → **完全无声**  
- 有文件优先；`select/invalid/ai` 与缺文件时走程序化兜底，**成品主声不得只剩蜂鸣感**  
- 首次用户手势后再解锁 AudioContext  
- 与画面 ~200ms 反馈大致对齐  

## 3. 用户期待

戴耳机默认音量不刺耳；连吃有节奏但不烦；静音键可信。

## 4. 目视/听感过线

换音后：戴耳机走一遍 step → jump → chain → win/lose；确认静音键全局静音。
