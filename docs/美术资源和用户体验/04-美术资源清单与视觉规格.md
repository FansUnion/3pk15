# 04 · 美术资源清单与视觉规格

> 用户期待：打开像「牧场猎杀棋」，不是后台工具或 Excel；**36px 仍能分出狼/羊/岩**。

## 1. 资源总表

| ID | 路径（约定） | 用途 |
|----|--------------|------|
| wolf-default | `public/skins/default/wolf.svg` | 默认狼 |
| sheep-default | `public/skins/default/sheep.svg` | 成套羊 |
| wolf-frost | `public/skins/frost/wolf.svg` | 霜狼 |
| sheep-frost | `public/skins/frost/sheep.svg` | 成套霜羊 |
| board-* | `public/skins/boards/{default,spring,summer,autumn,winter}.svg` | 棋盘底 |
| rock | 绘制于 BoardSvg 或 `public/skins/rock.svg` | 岩石 |
| hero-board | `public/brand/hero-board.*` | 首页主视觉 |
| mark | `public/brand/mark.svg` | 方标 |
| og | `public/og.png` 1200×630 | 分享 |
| icons | favicon / apple / 192 / 512 | 站点图标 |

岩石无皮肤目录热更；全局一式，可随季节微调色相。

## 2. 狼 · 必须长什么样

| 项 | 规格 |
|----|------|
| 造型 | 侧脸或 3/4；**耳、吻、肩**可读 |
| 色彩 | 暗色皮毛 + 少量高光；与羊明度分离 |
| 缩略 | viewBox 建议 `0 0 64 64`；在约 **36px** 仍辨认为狼 |
| 禁止 | 圆头+两点眼+三角耳的「通用动物」；与羊同色同明度 |

**用户期待**：选中时像「我的猎手」，不是色块贴纸。

## 3. 羊 · 必须长什么样

| 项 | 规格 |
|----|------|
| 造型 | 蓬松体块 + **深色脸/腿**对比 |
| 成套 | 与同套狼同一笔触、描边粗细 |
| 缩略 | 36px 仍辨认为羊 |
| 禁止 | 纯白无脸圆饼；与狼剪影雷同 |

## 4. 岩石

| 项 | 规格 |
|----|------|
| 造型 | 不规则多边形 + 2～3 面明暗/裂纹暗示 |
| 禁止 | Bootstrap 圆角棕方块 |
| 季节 | 可微出色温，保持「石」识别 |

## 5. 棋盘底板

| 项 | 规格 |
|----|------|
| 场感 | 草地/霜地纹理或轻 pattern；**交点比底更清晰** |
| 四季 | 春嫩绿 / 夏暖麦 / 秋琥珀 / 冬霜蓝——色温可辨 |
| 线 | 是路径不是 Excel；对比足够 |
| 禁止 | 大噪声糊线；四季只有同一灰绿换名 |

## 6. Hero / OG / Icon

- Hero：对局构图海报（三狼对羊群可读），非三枚散落小 icon  
- OG：Fangrush 字标 + 迷你盘；禁紫霓虹  
- Icon：简化狼吻/三狼点几何，小尺寸可辨  

## 7. Catalog 契约

- 存档只存 id；图鉴与对局 **同一 URL**  
- `wolf_set` 强制含羊资源；`pnpm check:skins` 必须过
