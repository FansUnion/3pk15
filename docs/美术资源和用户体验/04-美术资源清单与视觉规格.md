## 2. 狼 · 必须长什么样

| 项 | 规格 |
|----|------|
| 造型 | 侧脸或 3/4；**耳、吻、肩**可读 |
| 色彩 | 暗色皮毛 + 少量高光；与羊明度分离 |
| 缩略 | viewBox 建议 `0 0 64 64`；在约 **36px** 仍辨认为狼 |
| 禁止 | 圆头+两点眼+三角耳的「通用动物」；与羊同色同明度 |

## 3. 羊 · 必须长什么样

| 项 | 规格 |
|----|------|
| 造型 | 蓬松体块 + **深色脸/腿**对比 |
| 成套 | 与同套狼同一笔触、描边粗细 |
| 缩略 | 36px 仍辨认为羊 |
| 禁止 | 纯白无脸圆饼；与狼剪影雷同 |

## 4. 岩石

| 项 | 规格 |
|----|------|
| 造型 | 不规则多边形 + 2～3 面明暗/裂纹暗示 |
| 禁止 | Bootstrap 圆角棕方块 |
| 季节 | 可微出色温，保持「石」识别 |

## 5. 棋盘底板

| 项 | 规格 |
|----|------|
| 场感 | 草地/霜地纹理或轻 pattern；**交点比底更清晰** |
| 四季 | 春嫩绿 / 夏暖麦 / 秋琥珀 / 冬霜蓝——色温可辨 |
| 夜盘 | 深色夜场（Moonlit）与冬盘可分 |
| 线 | 是路径不是 Excel；对比足够 |
| 禁止 | 大噪声糊线；四季只有同一灰绿换名 |

## 6. Hero / OG / Icon

- Hero：对局构图海报（三狼对羊群可读），非三枚散落小 icon  
- OG：Fangrush 字标 + 迷你盘；禁紫霓虹  
- Icon：简化狼吻/三狼点几何，小尺寸可辨  

## 7. Catalog 契约

- 存档只存 id；图鉴与对局 **同一 URL**  
- `wolf_set` 强制含羊资源；`pnpm check:skins` 必须过  
- 详见 [`技术设计/04`](../技术设计/04-关卡存档与皮肤.md)

## 8. 目视过线（换皮后）
