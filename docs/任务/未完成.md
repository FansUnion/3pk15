## 生产环境变量（独立站）

模板：仓库根 [`.env.example`](../../.env.example)。

Vercel → Environment Variables（Production）：

```bash
NEXT_PUBLIC_SITE_URL=https://fangrush.com
NEXT_PUBLIC_APP_SHELL=standalone
NEXT_PUBLIC_ADS_PROVIDER=mock
ADMIN_ENABLED=true
ADMIN_ACCESS_KEY=请换成强口令
NEXT_PUBLIC_TELEMETRY=0
```

| 变量 | 说明 |
|------|------|
| `NEXT_PUBLIC_SITE_URL` | `metadataBase`；本地 `http://localhost:5000` |
| `ADMIN_ACCESS_KEY` | 公网非空强口令 |
| `NEXT_PUBLIC_ADS_PROVIDER` | 先 `mock`；有号后 `adsense` |
| `NEXT_PUBLIC_FAIL_ADS` | **生产勿设** |

GA4 测量 ID 写死在 `apps/web/src/components/Ga4.tsx`，**不需要** env。

### Vercel monorepo

1. Root Directory：`apps/web`  
2. Install：`cd ../.. && pnpm install`  
3. Build：`cd ../.. && pnpm --filter @wolf-sheep/web build`  
4. 域名：`fangrush.com`（Cloudflare DNS）

### 门户

`NEXT_PUBLIC_APP_SHELL=portal`、`ADMIN_ENABLED=false`、`NEXT_PUBLIC_TELEMETRY=0`；构建 `pnpm build:portal`。

### 上线冒烟

首页 → 春日可胜可败；刷新有进度；隐私页可开；`/admin` 有钥可进。

### 合规要点（一期）

无账号；棋谱/AI 不上自建服；门户默认零自建遥测；进度仅 localStorage。
