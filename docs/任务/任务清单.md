## 用户反馈方式

- 普通体验问题：提供页面、设备、操作和不舒服之处即可。
- AI、终局、循环或连吃疑问：优先点击“报告本局问题”并提供 JSON。
- 声音或审美：直接说明“听不清、像错误声音、认不出、哪部分难看”即可。

自动化通过不替代真实体验；人工感觉也不直接替代棋谱和固定种子证据。开发侧负责把两者合并成结论。
