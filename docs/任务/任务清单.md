## 两种发布形态（先看懂差别）

| | 独立站（fangrush.com） | 第三方小游戏平台（Poki / CrazyGames 等） |
|--|------------------------|------------------------------------------|
| 给谁玩 | 直接打开官网的玩家 | 平台里点进来的玩家 |
| 进度存哪 | 浏览器本地（现行）；账号云存档见 `P-ACCOUNT` | **只有**浏览器本地，无账号 |
| 广告 | 自己接 Google AdSense 等 | **必须用平台提供的广告接口** |
| 后台 `/admin` | 可以开 | **不能带**；打包时要剔掉 |
| 统计 | 可用自建 GA4（可关） | 默认不接自建远程统计 |
| 上架材料 | 官网自己运营 | 需要平台截图、简介、封面 |

两端共用同一套玩法与 `game-core`；差别只在壳、广告适配器、是否含 Admin。

**类型说明**

| 类型 | 含义 |
|------|------|
| 玩家功能 | 玩家在游戏里直接用到、看得见 |
| 广告商业化 | 赚钱、广告缝、上架变现相关 |
| Admin | 只有你（开发/运营）在独立站后台用 |
| 发布工程 | 挂站、打包、冒烟、环境变量 |
