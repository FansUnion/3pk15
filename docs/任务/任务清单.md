## 两种发布形态（先看懂差别）

| | 独立站（fangrush.com） | 第三方小游戏平台（Poki / CrazyGames 等） |
|--|------------------------|------------------------------------------|
| 给谁玩 | 直接打开官网的玩家 | 平台里点进来的玩家 |
| 进度存哪 | 浏览器本地（现行） | **只有**浏览器本地，无账号 |
| 广告 | 有量后再认真接 AdSense；没量先 Mock（不单列任务） | **必须用平台提供的广告接口**（冷启动主变现） |
| 后台 `/admin` | 可以开 | 门户构建已关（`pnpm build:portal`）；不单列任务 |
| 统计 | 可用自建 GA4（可关） | 默认不接自建远程统计 |
| 上架材料 | 官网自己运营 | 需要平台截图、简介、封面（`P-STORE`） |

两端共用同一套玩法与 `game-core`；差别只在壳、广告适配器、是否开 Admin。

| 类型 | 含义 |
|------|------|
| 玩家功能 | 玩家在游戏里直接用到、看得见 |
| 广告商业化 | 赚钱、广告缝、上架变现相关 |
| Admin | 只有你在独立站后台用 |
