# Fangrush 当前工作总览

> 当前状态以 [任务清单](任务清单.md) 为准。本文只说明目标、边界和入口。

已经关闭的地图、AI、体验与发布批次计划已归档。羊 AI V2 已完成自动开发与验证；当前只剩[人工验收](剩余人工验收与文档治理.md)和双 H5 平台基础材料。

## 当前阶段

玩家端与 Admin 已拆为两个独立应用并部署到 Vercel：

- 玩家端：`https://fangrush.com`
- Admin：`https://admin.fangrush.com/admin`
- Admin 功能统一位于 `/admin/*`；玩家端不展示 Admin 入口。

代码侧发布基线、地图重建、玩家全旅程和羊 AI V2 确定性开发已经完成。2026-07-19 的 720 局固定种子矩阵关闭了 `winter-04 LONG_TAIL`，24关当前为 `15 pass / 9 review / 0 reject`；`review` 只进入定向人工抽查，不阻塞生产体验。

## 三条近期主线

### 1. 生产环境体验验收

第二、三批确定性问题已经修复并通过自动回归。用户继续在真实手机、平板和桌面体验玩家端与 Admin；局面、AI、终局或声音问题优先通过玩家端“报告本局问题”导出 JSON，Admin 可直接导入和回放。

### 2. 羊 AI 与24关验收

地图重建、新鲜度门禁、24关羊决策质量审计和五级 AI 画像已经完成。玩家只需抽查长尾、首次接触和冬季专家防守关；发现异常后由开发侧用问题包复现。

### 3. 平台基础材料

近期只准备 Poki early access 和 CrazyGames Basic 首次提交需要的资料，包括字段口径、文案、截图、封面、视频和主体信息清单。真实 SDK、广告、生命周期、Inspector/Portal 联调不是当前任务，统一保留在需求池。

## 当前边界

- 24关资料卡、AI画像和重建基线已具备；当前自动诊断为 `15 pass / 9 review / 0 reject`，玩家人工验收不等同于该自动诊断。
- 平台本地构建边界和合规矩阵已具备，但不表示平台审核或真实 SDK 已完成。
- 已完成事项不继续堆积在任务清单；长期功能不进入当前执行队列。
