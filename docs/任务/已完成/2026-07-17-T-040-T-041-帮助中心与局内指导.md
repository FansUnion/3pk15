# T-040 / T-041 帮助中心与局内指导

完成日期：2026-07-17

## 完成范围

- `T-040`：现有 `/how-to-play` 扩展为帮助中心，包含基础规则、操作反馈、狼方策略和移动/隔空吃 ASCII 图例；主要页面沿用统一帮助入口。
- `T-041`：正式对局增加不离开棋局的帮助弹窗，以及“观察、目标、策略”三级主动提示；连续失败两次后提示可求助。
- 24 关指导复用 `LevelConfig` 的教学点、玩家目标和狼方策略，不维护第二套关卡答案。
- 存档记录每关提示使用次数和连续失败次数；旧存档迁移时自动补齐，胜利后清零该关失败连击。

## 边界

本批不提供自动落点、完整路线演示、悔棋或广告。指导帮助玩家观察和规划，但不替玩家执行行动，也不用于掩盖待修订地图。
