# 任务库说明

[任务清单](任务清单.md) 只维护当前执行工作；[需求池](需求池.md) 维护暂缓功能；[已完成](已完成/README.md) 保存关闭任务的必要证据。`P-ACCOUNT.md` 是需求池中账号与云存档的详细附件。

- 改产品目标、规则、AI、布局或平衡目标：先更新 [核心产品需求与质量要求](../产品核心/00-核心产品需求与质量要求.md)。
- 新想法：先进入 [需求池](需求池.md)。
- 决定马上开发：从需求池迁入 [任务清单](任务清单.md)。
- 完成任务：从任务清单移除，必要证据迁入 [已完成](已完成/README.md)。
- 未来关卡、赛季和挑战玩法：维护在 [产品扩展](../产品扩展/README.md)。
