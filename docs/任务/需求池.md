# Fangrush 需求池

> 本文维护已经提出、但依赖玩家数据、平台结果或额外投入才能启动的功能。近期执行只看 [任务清单](任务清单.md)；未来关卡和玩法内容见 [产品扩展](../产品扩展/README.md)。

## 使用规则

- 需求池不是开发承诺，也不阻塞当前双H5首发。
- 每项必须有价值、风险和启动条件；条件满足后迁入任务清单，不并行维护状态。
- 影响核心操作的UI、基础无障碍和奖励表达已进入 `T-044/T-016`，不再作为暂缓项。

## 第五批：真实数据驱动的体验增强

以下能力在H5上线并获得玩家行为或访谈证据后决定。

### 悔棋

- 价值：降低误操作和长局单步失误造成的挫败。
- 风险：必须完整恢复羊方AI随机状态、连吃起点、局面计数和奖励边界。
- 启动条件：玩家验证或线上数据证明误操作是主要失败与流失原因。

### 高级提示与路线演示

- 价值：帮助多次失败但愿意继续的玩家跨过策略理解门槛。
- 风险：模拟策略不是必胜答案，不能用提示掩盖坏地图。
- 启动条件：当前三级指导仍无法解决可复现的策略理解问题。

### AI节奏与难度选项

- 价值：允许玩家调整等待时间或临时降低压力。
- 风险：不能改变主线官方平衡数据和默认AI标准。
- 启动条件：玩家数据证明默认AI节奏或压力阻碍核心体验。

### 独立站账号与统一云存档

- 价值：跨设备同步、恢复进度和未来统一运营。
- 风险：引入后端、认证、隐私、安全、迁移和长期运维成本；平台账号能力不完全一致。
- 启动条件：出现明确跨设备需求，或多个渠道需要统一身份与进度。
- 详细方案：[P-ACCOUNT](P-ACCOUNT.md)。

### 专业美术、角色动画与原创音频

- 价值：提高第一印象、商店点击、角色辨识、操作反馈和品牌长期价值。
- 当前边界：T-044负责可上线的一致性、基础动效和平台封面；本项负责品牌级重制、完整状态帧、BGM与环境音。
- 启动条件：H5数据证明值得扩大投入，或玩家/平台反馈指出当前资源明显限制转化与留存。

### 高级无障碍

- 范围：完整读屏、全键盘棋盘操作、高对比主题、细粒度音量和更系统的可访问性测试。
- 启动条件：基础发布门禁通过，或目标平台与真实用户提出明确需要。

### 奖励与长期留存深化

- 范围：皮肤主题故事、奖励曲线实验、重复游玩收益、收藏展示和更长期目标。
- 启动条件：上线数据证明奖励理解、回访动力或收藏价值是主要问题。

### 分享增强与传播实验

- 范围：完整棋谱链接、挑战好友、分享模板A/B测试和平台传播活动。
- 当前边界：基础结果卡、局面图和系统分享已进入 `T-044`。
- 启动条件：基础分享产生可观察使用量或传播价值。

## 商业化与增长实验

### H5广告与商业漏斗优化

- 当前边界：Poki/CrazyGames基础广告SDK接入属于 `T-017`。
- 后续范围：广告机会点、激励价值、留存、广告完成率和收入的实验优化。
- 启动条件：平台上线并获得足够真实流量；不得以收入提升掩盖首局完成或留存下降。

## 第六批：App商业化与多平台扩张

以下能力只有在至少一个H5平台取得可接受的点击、首局完成、游玩时长、留存和广告数据后启动。

### Google Play与App Store

- Capacitor或其他App壳、移动端生命周期、离线与升级策略。
- AdMob等原生插屏和激励广告。
- 去广告IAP、付费皮肤IAP、恢复购买和交易幂等。
- 商店账号、隐私披露、数据安全、年龄分级、审核说明和商店素材。
- 崩溃监控、版本回滚、推送和基础商店运营。

### 其他渠道

- Steam：仅在内容达到买断级规模和深度后评估。
- Windows/Mac商店：不单独作为近期目标，可随App能力顺带评估。
- 独立站广告：当前保持无广告Demo；有稳定自有流量后再决定是否接入。
