# 指标、实验与决策规则

## 核心指标

- 首屏行动率：打开首页后点击开始或继续的比例。
- 首次有效操作率：进入棋局后完成第一步有效操作的比例。
- 首局完成率：开始第一局后到达胜利或失败结果页的比例。
- 首次胜利率：新玩家在规定尝试内完成首胜的比例。
- 失败重试率：失败后重新开始当前关卡的比例。
- 第二次进入率：首次离开后再次打开游戏的比例。
- 皮肤转化率：进入皮肤页后完成预览、解锁或装备的比例。
- 广告自愿完成率：主动选择广告且成功获得奖励的比例。

## 实验原则

1. 一次只改变一个主要变量，例如引导方式、第一关布局或奖励数量。
2. 先验证理解和留存，再优化广告收入。
3. 不用收入提升掩盖首局完成率下降。
4. 所有实验都必须可以关闭，不把实验逻辑写死在核心规则中。
5. 任何影响公平性的变化必须经过规则测试和人工复盘。

## 决策优先级

- P0：无法开始、无法操作、状态丢失、奖励错误、广告阻塞玩法。
- P1：核心路径困惑、明显不公平、移动端不可用、关键资源无法识别。
- P2：页面层级、文案、节奏、动效和展示质量问题。
- P3：高级优化和个性化体验。

优先修复影响“第一次有效操作”和“第一次完成一局”的问题，再处理皮肤深度、长期任务和广告收益。

## 当前阶段结论

当前最合理的顺序是：先完成基线审计和首局体验，再打磨关卡平衡与资源识别，之后完善长期奖励，最后接入商业化实验。商业化不是独立模块，而是建立在玩家已经理解并愿意重复游玩的核心体验之上。
