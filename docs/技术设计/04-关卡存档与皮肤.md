## LevelConfig（只读，进仓库）

关键字段：`id`、`chapterId`、`rocks[]`、`ai`、首次/重复掉落；展示/SEO：`nameZh`/`nameEn`、`blurbZh`/`blurbEn`（`name` 与 `nameZh` 同步）。

**校验**

- 岩石在盘内、不重叠、**四邻不相邻**、不占开局子位。  
- 春 0/1/2 岩；夏 2–4；秋 6–8；冬 0。  
- `ai` 与章绑定：春 easy；夏/秋 normal；冬 hard。

**解锁**：春默认开；通关本章全部 level → 解锁下一章。

## SaveGame

关键字段：`schemaVersion`、`clearedLevels`、`unlockedChapters`、`fragments`、`unlockedSkinIds`、`equipped`、`guide`、`buffs.doubleDropUntil`、`lastPlayedLevelId`。

- 存储键经 `IStorage`（如 `wolf-sheep-save-v1`）。  
- 读写失败 → 默认档。  
- 迁移：`migrate(raw)`；升版本禁止静默丢字段。  
- 默认：解锁 spring；default 狼套 + 默认棋盘；碎片 0。

**后台不写 SaveGame**；模拟结果默认不落盘。

## 皮肤 Catalog

| kind | 说明 |
|------|------|
| `wolf_set` | 狼 + **成套羊**（强制） |
| `board` | 棋盘主题；与狼皮独立 |
| 岩石 | 无皮肤 |

- 存档只存 id；资源包内。  
- `check:skins`：文件存在、id 唯一、至少一套 default；缺羊 → CI 失败。  
- `resolveSkin`：equipped 失效 → 回退 default。  
- **不做**远程热更；改皮随发版。

视觉气质（呈现层）：牧场浅色、剪影可辨；细则见创意 08，不在此展开页稿。
