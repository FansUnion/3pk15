# 04 · 关卡存档与皮肤

> 关卡意图：创意 [04](../游戏创意/04-岩石地形与难度梯度.md)、[05](../游戏创意/05-四季关卡设计.md)。  
> 碎片皮肤任务：创意 [07](../游戏创意/07-内容进度-碎片皮肤任务.md)。  
> **真相源代码**：`packages/game-core/src/content/levels.ts`、`save.ts`、`skins.ts`。

## 必须怎样

- 关卡只读进仓库；岩石校验：盘内、不重叠、四邻不相邻、不占开局子位；章节岩石区间与 `ai` 档与创意一致。
- 解锁：通关本章全部 level → 下一章。
- 存档经 `IStorage` + `schemaVersion` + `migrate`；读写失败回默认档；升版本禁止静默丢字段。
- 皮肤 Catalog：`wolf_set` 强制含成套羊；`board` 独立；岩石无皮肤。
- `check:skins` 过 CI；`resolveSkin` 失效回退 default。

## 不许怎样

- 后台写玩家 `SaveGame`；模拟结果默认不落盘。
- 远程热更皮肤（改皮随发版）。
- 存档存资源 URL（只存 id）。

字段清单与默认值以代码为准，本文不抄写。视觉气质见 [`docs/美术资源和用户体验/`](../美术资源和用户体验/00-索引.md)。
