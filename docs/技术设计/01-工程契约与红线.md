## 工程边界

| 必须 | 禁止 | 真相源 |
|------|------|--------|
| 玩法在 `packages/game-core`；壳只调 API | core 依赖 DOM / Next / zustand / 广告 SDK / 直接读 `localStorage` | `packages/game-core`、`apps/web` |
| 差异进 Adapter（`IStorage` / `IAds` / `IAdminGate`） | `if (portal)` 散落规则层；结算直接调第三方全局对象 | `apps/web` adapters |
| 对局 AI 在浏览器跑 | AI 放 Server Route；admin 另写一套 AI | `pickSheepAction` |
| 存档经 `IStorage` + `schemaVersion` + migrate | 无迁移的破坏性改结构；读写失败不回退 | `content/save.ts` |
| CI：lint + game-core test + `check:skins` + web build | 无 fixture 合入规则/AI 行为变更 | 仓库 scripts / CI |

**选型（定稿）**：TS strict · Next App Router · React + Tailwind · 棋盘 SVG · pnpm。  
**部署优先级**：P0 独立站 + `/admin` → P1 经济/广告 → P2 门户瘦包 → App 端口预留。账号暂缓见 [`P-ACCOUNT`](../任务/P-ACCOUNT.md)。

| 端 | `/admin` | 广告 | 遥测 |
|----|----------|------|------|
| 独立站 | 可开 | AdSense / Mock | 可后接 |
| 门户 | 必须无 | 门户 SDK / Mock | **默认零自建远程** |

关键 env 类别：`NEXT_PUBLIC_APP_SHELL`、`NEXT_PUBLIC_ADS_PROVIDER`、`ADMIN_ENABLED`（完整键名见 `apps/web/.env.example`；打包防呆见 [07 §3.3](./07-游戏架构总览.md)）。  
IAds：`showInterstitial` 失败也继续；`showRewarded` **仅 ok** 发奖。产品口径 → 创意 [09](../游戏创意/09-广告与变现设计.md)。

## 规则

| 必须 | 禁止 | 真相源 |
|------|------|--------|
| 着法唯一入口：`listLegalActions` → `applyAction` | UI / Admin 自写吃子或胜负 | `rules.ts`；意图 [创意 02](../游戏创意/02-核心对局规则.md) |
| 连吃 ≤5；有吃可不强制；无 jump 自动结束；UI 可 `endWolfTurn` | 斜走、穿岩、同点叠子 | 同上 |
| 胜：`eatenSheep ≥ 8`；败：三狼皆无合法着 | 非 `playing` 仍接受对局着法（reset 除外） | 同上 |
| 羊每回合一步、禁向第 1 行后退 | AI 搜索改全局单例（须不可变或显式 clone） | 同上 |

改规则必须带 Vitest fixture（开局着法、隔空吃、岩挡、连吃切羊、吃满 8、羊禁后退、三狼堵死、非法 apply、随机不变量等）。

## AI

| 必须 | 禁止 | 真相源 |
|------|------|--------|
| 统一入口 `pickSheepAction(state, { difficulty, rng, budgets? })` | Admin / 模拟台绕开该函数 | `packages/game-core/src/ai/`；意图 [创意 06](../游戏创意/06-AI行为设计规格.md) |
| 章默认档：春 easy、夏/秋 normal、冬 hard | 按玩家胜率动态改档；深度学习或服务端另写 AI | 同上 |
| 同 state + 同 RNG 种子 → 同着法；hard 降级可观察 | 把未冻结的权重/胜率表写成「技术定稿」 | `ai-behavior` 等测试；平衡见任务清单 |

**Fixture 红线（工程）**：唯一挡吃 → normal/hard 必挡；送吃诱惑 → normal/hard 不送；预算耗尽 → hard 降级；冬日合围意图仍以任务 `P-AI-FIX` 验收为准。算法细节读代码，不在此抄写。

## 关卡 / 存档 / 皮肤

| 必须 | 禁止 | 真相源 |
|------|------|--------|
| 关卡只读进仓库；岩石：盘内、不重叠、四邻不相邻、不占开局子 | 后台写玩家 `SaveGame`；模拟默认落盘 | `content/levels.ts`、`save.ts`；意图创意 04/05/07 |
| 通关本章全部 level → 解锁下一章 | 远程热更皮肤（改皮随发版） | 同上 |
| Catalog：`wolf_set` 含成套羊；`board` 独立；岩石无皮肤 | 存档存资源 URL（只存 id） | `content/skins.ts`；`check:skins` |
| `resolveSkin` 失效回退 default | — | 同上 |

字段与默认值以代码为准。视觉规格 → [`美术资源和用户体验/`](../美术资源和用户体验/00-索引.md)。

## Admin（Content & Quality Cockpit）

独立站 `/admin`：开发者/内容质量驾驶舱，**不是** CMS / 用户后台 / 远程看板。`ADMIN_ENABLED=false` → 404；门户构建剔除。

| 台 | 一句话 |
|----|--------|
| 皮肤 | 身份感与资源齐套验收 |
| 关卡 | 布局合法 + blurb / SEO 文案 |
| AI 模拟 | 与线上同 `pickSheepAction`；粗校准（狼方随机合法着） |
| 玩法说明 | 与 `/how-to-play` 同源 |
| 经济 | 碎片→皮肤闭环只读表 |
| 上线清单 | 漏斗断点本机勾选 |

**硬约束**：只读 Catalog / LEVELS / QUEST + 静态资源；**不写** SaveGame；不做写库、账号、远程漏斗、真广告控制台。
