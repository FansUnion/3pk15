# 02 · 视觉与音效资源规格

> **体验承诺**：打开像「牧场猎杀棋」；约 **36px** 仍能分出狼 / 羊 / 岩。普通走子轻、隔空吃与连吃有层次、胜负短促不吵。  
> Catalog：[`packages/game-core/src/content/skins.ts`](../../../packages/game-core/src/content/skins.ts)。音效入口：[`apps/player-web/lib/sfx.ts`](../../../apps/player-web/lib/sfx.ts)。
> 资源源目录为 `packages/web-assets/public/skins/`、`packages/web-assets/public/brand/`、`packages/web-assets/public/sfx/`，构建时同步到 `apps/player-web/public/`；玩家运行时只从 `apps/player-web` 读取。CI：`pnpm check:skins`。
> 专业状态帧、环境音和主题 BGM 尚未启动，统一见 [需求池](../../任务/需求池.md)。

## 气质

| 主题 | 气质 |
|------|------|
| 默认 / 原野 | 暖日光草地，清晰交点，朴实可读 |
| 霜 (frost) | 冷白高光、皮毛更硬，不洗成一片灰 |
| 夜 (night) | 深蓝月夜场，狼羊仍高对比，非纯黑糊 |

## 库存（以 Catalog 为准）

当前入仓：**3 套狼羊**（default / frost / night）+ **6 块棋盘**（default、春夏秋冬、night）+ brand（hero-board、mark、og、icons）。  
**岩石**：无独立皮肤文件；`BoardSvg` 的 `RockShape` 程序绘制，可按棋盘皮微调色温。  
解锁价与路径以 `skins.ts` / `public/` 为准，本文不抄价。

## 狼 / 羊 / 岩 / 棋盘 · 必须长什么样

| 资产 | 必须 | 禁止 |
|------|------|------|
| 狼 | 侧脸或 3/4；耳、吻、肩可读；暗色皮毛 + 少量高光；约 36px 仍辨认为狼 | 圆头+两点眼+三角耳；与羊同色同明度 |
| 羊 | 蓬松体块 + 深色脸/腿；与同套狼同一笔触；36px 仍辨认为羊 | 纯白无脸圆饼；与狼剪影雷同 |
| 岩石 | 不规则多边形 + 2～3 面明暗/裂纹 | Bootstrap 圆角棕方块 |
| 棋盘 | 草地/霜地纹理；交点比底更清晰；四季色温可辨；夜盘与冬盘可分 | 灰线 Excel 盘；四季同色换名；紫霓虹边框 |

viewBox 建议 `0 0 64 64`。Hero：对局构图海报，非散落小 icon。OG：Fangrush 字标 + 迷你盘。Icon：简化狼吻/三狼点，小尺寸可辨。

## 音效

以 `sfx.ts` 事件名为准。

| 事件 | 文件 | 听感 | 时长 | 忌 |
|------|------|------|------|-----|
| step | `public/sfx/step.wav` | 轻木棋「笃」 | < 120ms | 手机按键音 |
| jump | `public/sfx/capture.wav` | 短 whoosh + 轻咔 | < 200ms | 枪声/血腥 |
| chain | `public/sfx/chain.wav` | 同族音高递进 | < 200ms | 过长旋律 |
| win | `public/sfx/win.wav` | 短上行解决 | < 500ms | 长 fanfare |
| lose | `public/sfx/lose.wav` | 短下行 | < 400ms | 刺耳报警 |
| select / invalid / ai | （无 wav） | 程序化轻/弱提示 | — | 抢对局声 / 刺耳 / 像 loading |

**契约**：`settings.muted === true` → 完全无声；有文件优先，缺文件走程序化兜底但成品主声不得只剩蜂鸣；首次用户手势后再解锁 AudioContext；与画面 ~200ms 反馈大致对齐。

| 层 | 例子 | 期望 |
|----|------|------|
| 操作 | step / select | 轻、短、可连点不烦 |
| 规则反馈 | jump / invalid | 比操作更醒目一层 |
| 情绪 | chain / win / lose | 连吃递进；胜负短促有收束 |

听感过线：戴耳机走一遍 step → jump → chain → win/lose；确认静音键全局静音。
