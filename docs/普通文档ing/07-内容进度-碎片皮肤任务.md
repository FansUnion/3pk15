# 07 · 内容进度：碎片 · 皮肤 · 任务

玩法经济设计（不含广告触发细节，广告见 09）。

## 碎片种类

| 类型 | 用途 |
|------|------|
| 通用万能碎片 | 兑换稀有皮肤（如霜狼等） |
| 四季专属碎片 | 兑换对应季节棋盘皮肤 |

## 碎片获取

1. **首次通关**关卡：固定保底掉落  
2. **重复刷**已通关关卡：概率少量掉落  
3. **每日 / 每周任务**：完成领取  
4. **激励广告**：补碎片 / 双倍掉落时长（规则见 09）

通关时发放基础碎片。

## 皮肤构成

1. **狼皮肤**（核心收集）：三只狼统一外观  
2. **羊皮肤**：跟随当前狼皮肤 **成套自动切换**，不单独兑换  
3. **棋盘皮肤**：四季主题 + 限定棋盘，独立解锁/切换，与狼皮不绑定  
4. **岩石**：无皮肤  

### 解锁方式

- **基础**：通关章节直接解锁（四季棋盘、原生狼羊外观）  
- **稀有**：消耗通用碎片合成（霜狼等）  

### 穿戴

- 换狼皮 → 场上羊外观同步成套  
- 棋盘皮可单独切换  

## 留存设计意图

- 老玩家需 **回刷春日低难度快局** 攒碎片，缓解「通关即流失」
- 秋日为高效刷碎片的爽关时段（见 05）

- 独立站与第三方平台：均为浏览器本地缓存碎片、已解锁皮肤、关卡进度（与 [09](./09-广告与变现设计.md) 一致）
