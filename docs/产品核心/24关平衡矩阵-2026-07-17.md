# 24 关固定种子双策略平衡矩阵（2026-07-17）

> 用途：完成 `T-003` 的生产风险筛选。模拟结果不是玩家胜率，也不代替人工试玩。

## 复现口径

- 方法基线：T037/T038 修复后的候选门禁；提交见本文件 Git 历史。旧 `940deaf` 数据仅作为历史比较，不再代表当前结论。
- 羊 AI：使用各关正式 `ai`；hard 固定 `maxNodes=80`、无时间上限，并只在 Normal 即时最优动作集合中做有限预测。
- 狼策略：`random`（合法行动均匀随机）与 `mixed`（65% 选择即时评分最优行动，35% 随机探索）；mixed 在存在新局面行动时避开重复。
- 随机流：狼与羊由同一种子派生独立 RNG，AI 档位或搜索节点消耗不会改变狼的探索序列。
- 种子：`20260717..20260726`；每关每策略 10 局，共 `24 * 2 * 10 = 480` 局。
- 指标：`W/S/D` 为狼胜/羊胜/和棋；`Avg/P95` 为平均/P95 plies；`Eat` 为平均捕食数；`Cap` 为发生过捕食的局数（满值 10）。
- 终局：`T` 目标捕食、`B` 狼无合法行动、`R` 三次重复、`M` 达到 `maxPlies`。

复现命令（`<season>` 依次为 `spring`、`summer`、`autumn`、`winter`）：

```powershell
$env:RUN_BALANCE_MATRIX='1'
pnpm --filter @wolf-sheep/game-core exec vitest run tests/all-level-balance-matrix.test.ts -t "<season> two-strategy" --reporter=verbose
```

## 数据支持的结论

1. 24 关都可用正式初始化、正式规则和正式羊 AI 完成固定种子模拟，没有 `unexpected` 终局或执行异常。
2. 狼策略质量显著影响结果：mixed 在全部关卡都优于 random，说明游戏存在可执行的策略收益，而非纯随机胜负。
3. 羊方主要防守结果是狼无合法行动；冬季 mixed 的 60 局中羊方取得 10 局困狼胜、没有重复和棋。冬季已从“只会拖和”转为真实困狼压力，但策略收益很高，仍需验证玩家是否能理解胜线。
4. spring-01 与 spring-02 原配置实际重复；spring-02 岩石已移至 `(4,4)`，修订后矩阵明确分化，仍需新手确认岩石作用是否可感知。
5. `maxPlies` 仅出现 3 局（spring-03 mixed、spring-04 random、winter-03 random 各 1 局），不是主要终局；反重复代理策略修复后，mixed 的重复显著下降。
