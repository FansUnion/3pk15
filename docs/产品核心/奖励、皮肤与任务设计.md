# 奖励、皮肤与任务设计

本文维护主线奖励、皮肤和任务的产品关系。具体库存、掉落参数和存档字段以代码为准；商业化触发不在本文定义。

## 碎片种类

| 类型 | 用途 |
|------|------|
| 通用万能碎片 | 兑换稀有皮肤（如霜狼等） |
| 四季专属碎片 | 兑换对应季节棋盘皮肤 |

## 碎片获取

1. **首次通关**关卡：固定保底掉落  
2. **重复刷**已通关关卡：概率少量掉落  
3. **每日 / 每周任务**：完成领取  
4. **未来可选激励**：是否通过平台广告补充碎片，须在真实商业实验中单独验证，不属于首发核心奖励。

通关时发放基础碎片。

## 皮肤构成

1. **狼皮肤**（核心收集）：三只狼统一外观  
2. **羊皮肤**：跟随当前狼皮肤 **成套自动切换**，不单独兑换  
3. **棋盘皮肤**：四季主题 + 限定棋盘，独立解锁/切换，与狼皮不绑定  
4. **岩石**：无皮肤  

### 解锁方式

- **基础**：通关章节直接解锁（四季棋盘、原生狼羊外观）  
- **稀有**：消耗通用碎片合成（霜狼等）  

### 穿戴

- 换狼皮 → 场上羊外观同步成套  
- 棋盘皮可单独切换  

## 留存设计意图

- 老玩家需 **回刷春日低难度快局** 攒碎片，缓解「通关即流失」
- 秋日为高效刷碎片的爽关时段（见 05）

## 存档原则

- 首发以本地进度满足单机体验；具体载体与字段由代码维护。
- 跨设备云存档属于 [需求池](../任务/需求池.md)，不改变现行奖励设计。

节日限定皮等功能资源需求见 [需求池](../任务/需求池.md)；每日挑战专属碎片随未来周期关卡在 [产品扩展](../产品扩展/README.md) 维护。
