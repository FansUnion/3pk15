# 2026-07-20 · 羊 AI V4 主动对手验证（历史）

> 本文保存 `sheep-ai-v4` 当时的三维对手合同、720局候选门禁和960局四类玩家代理矩阵，不代表当前生产结论。当前证据见[羊 AI V5 收口验证](2026-07-20-羊AI-v5收口验证.md)。

## 一、结论

- 24关均已接入独立的能力阶段、主要/辅助行为风格和本关意图；正式游戏、Admin、候选模拟和玩家代理统一从关卡配置读取。
- 720局候选门禁结果为 `9 自动未发现风险 / 15 需要复核 / 0 自动拒绝`；15个复核由9个首次捕食风险、4个长尾和2个固定猎手风险构成，不表示15张地图不合格。
- 全部720局未出现被支配羊步、可严格避免的长链暴露或搜索降级。新增门禁没有发现高无进展、低目标持续或低风格兑现关卡。
- 候选局面的主动压力率为65.5%–95.1%，无进展率为1.6%–9.0%；目标持续和风格兑现已经可以量化，但仍不能证明玩家一定能从视觉上读懂计划。
- 战术交换不再是死代码：`autumn-03`候选矩阵记录11次、玩家画像矩阵记录9次受控交换；每次最多增加1只羊的即时暴露，并立即增加被困狼数量。其他关卡的辅助交换风格不能突破安全底线。
- 960局玩家代理全局呈现合理技能增益：新手、普通、熟练、资深分别获得138、181、213、223局狼胜；被支配羊步和搜索降级均为0。
- 自动侧未发现需要立刻修改地图、开局或规则的确定性缺陷。剩余问题是少量真人才能判断的压力、反制可读性、固定猎手是否有趣，以及长局是否疲劳。

## 五、最多8局定向体验

| 关卡 | 只需判断的问题 |
|---|---|
| spring-01 | 新手是否被鼓励学习三狼协作，还是一只狼重复得分就足够 |
| spring-06 | 春季综合关的长局是否仍有清楚变化和收束感 |
| summer-03 | 漏斗首吃是否可读；失败能否归因于三狼分工 |
| summer-05 | 固定猎手连续得分时，羊的合围反制是否可信 |

| autumn-04 | 双岛换区是否有清楚入口，还是必须试错才能找到首吃 |
| winter-02 | 长尾过程是否持续有压力和反制，而不是双方往返 |

每局只记录“有灵性、可反制、有压力但可赢、只会拖延、过难”及一句原因。出现单步疑问时导出本局JSON，由代码侧复放，不要求重打一关。
