# 2026-07-20 · 羊 AI V3 与玩家画像验证

> 本文是 `sheep-ai-v3` 当前生产 AI、历史问题包、720 局候选门禁和 192 局玩家画像矩阵的日期型证据。它证明自动红线与策略样本，不等于真人胜率或 24 关最终产品验收。

## 一、结论

- 设定与实现目前为**条件可行**：24 关均未命中结构、异常终局、严格劣着或搜索降级红线，可进入最多 8 局定向真人体验。
- V3 候选门禁为 `12 自动未发现风险 / 12 需要复核 / 0 自动拒绝`。`review` 表示代理触发长尾或首次捕食风险，不表示地图不合格；`pass` 也不表示真人已经认可。
- 四类独立玩家代理表明：熟练和资深策略在大多数关都有狼方胜路，但策略效果并非简单线性。某些局面中固定猎手套路不如换线策略，说明单一代理不能代表玩家。
- 720 局候选门禁中 `chosenDominated=0`、严格定义的可避免长链暴露 `0`、搜索降级 `0`。192 局画像矩阵中严格劣着 `0`、搜索降级 `0`。
- 历史 `winter-06` 问题包在当前 V3 下为 `41/42` 个羊动作保持一致；唯一改变正是旧版第 54 个报告动作对应的即时暴露步。当前 V3 选择零即时暴露替代。
- 离线教师对旧争议步返回 `unknown`：20,000 节点未完成全部候选，当前可见后悔值为 16。它支持“旧动作可疑、V3 修改合理”，但不宣称已经证明理论最优。

## 三、V3 改变了什么

1. 生产评分新增终局紧迫度、固定猎手风险和目标压力，且在候选筛选阶段生效，不再等搜索后才看到风险。
2. `strategic/expert` 在存在非劣、最低即时捕食链候选时，不再主动选择更高即时暴露动作；未来只有教师证据能支持放宽。
3. Expert 提升到三层、4000 节点的确定性受限搜索。它仍不是全局求解器。
4. 新增四类独立狼方代理：新手、普通、熟练、资深。它们不调用羊 AI 的评分函数，避免同一模型验证自己。
5. 新增离线教师、问题包审计器、版本化反例样本、猎手集中度与连续猎手指标，以及 12 个两关学习阶段合同。
6. Admin 可直接显示玩家画像含义、捕食归属、猎手集中度、动作质量、搜索完成度和教师结论。

## 五、192 局玩家画像矩阵

以下为每季 6 关、每画像 2 种子合计 12 局的狼胜/羊胜/和。样本用于比较策略与阶段，不是统计意义上的玩家胜率。

| 季节 | 新手 | 普通 | 熟练 | 资深 |
|---|---:|---:|---:|---:|
| 春 | 7/5/0 | 10/2/0 | 12/0/0 | 12/0/0 |
| 夏 | 7/5/0 | 10/2/0 | 11/1/0 | 9/3/0 |
| 秋 | 3/8/1 | 8/3/1 | 10/2/0 | 11/1/0 |
| 冬 | 7/4/1 | 7/5/0 | 12/0/0 | 11/1/0 |

得到的产品判断：

- 春季整体可入门，但 `spring-02` 对所有代理都偏容易，需真人确认长局是否仍有教学价值；`spring-03` 普通代理失败而熟练代理全胜，适合检查学习跃迁是否可理解。
- 夏季开始明显策略敏感。`summer-01` 普通/熟练胜而资深固定套路失败，说明玩家需要换线而不是机械坚持单一猎手；这不是“专家代理一定更强”的证明。
- 秋季最符合学习曲线：新手胜 3/12、普通 8/12、熟练 10/12、资深 11/12，策略学习收益最清楚。
- 冬季对普通代理形成压力，但熟练/资深代理几乎全胜。当前证据支持“有挑战但存在胜路”，仍需真人判断胜路是否清楚、羊是否显得聪明，而不是只看结果。
- 较高即时链暴露次数从新手 39、普通 33、熟练 27 降到资深 18。该指标只是广义风险信号；只有严格替代条件或教师裁决才能称为可避免错误。

| 严格劣着 | 0 |
| 旧版即时暴露争议 | 1 次，ply 53 后选择 `sheep-11 -> 4,2` |
| V3 行为 | 改选零即时暴露动作 |

结论不是“一只狼不能吃多只羊”。规则允许固定猎手，另外两只狼可能负责压缩空间。需要拒绝的是“羊有明显安全替代却反复放任同一路线”。V3 已关闭本问题包中唯一确定的即时暴露差异；跨回合是否仍有更深反例，继续由教师和定向真人体验共同判断。

## 七、最多 8 局真人验收

自动侧前五批已完成。制作人只需各季两局，不再重跑 24 关：

| 关卡 | 只需判断什么 |
|---|---|
| spring-03 | 从基础移动进入双狼配合时，胜路是否自然可学，失败是否知道原因 |
| spring-06 | 春季考试是否仍宽容，但不靠羊明显送子 |
| summer-01 | 长局是否持续有进展，换线是否比固定套路更合理 |
| summer-06 | 首次突破是否看得懂，单猎手路线是否会被羊主动拆解 |
| autumn-04 | 岩石分区下是否有压力但不僵硬，较长对局是否仍有决策变化 |
| autumn-06 | 季末综合策略是否成立，三狼分工是否真实有用 |
| winter-02 | 长尾是否来自有价值攻防，而不是羊只会拖延 |
| winter-06 | 终关羊是否像专家防守；同一狼反复得分时是否有合理封堵反应 |

每局只需反馈“太笨 / 合理 / 有压力但可赢 / 只会拖延 / 过难”之一。遇到具体疑问再导出“报告本局问题”JSON，代码侧负责复现和归因。
