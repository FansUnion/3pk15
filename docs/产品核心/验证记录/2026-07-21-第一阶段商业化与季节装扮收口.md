## 实现结论

| 项目 | 结果 |
|---|---|
| 通用碎片目标 | 霜狼 80、夜巡狼群 160、极光盟誓 240 |
| 无广告首次通关路径 | 24 关获得 240；第 8 关兑换霜狼，第 24 关兑换夜巡狼群 |
| 一半首通观看路径 | 12 次广告、共 360；完成前两套，继续推进极光盟誓 |
| 全部首通观看路径 | 24 次广告、共 480；第 4/12/24 关完成三套 |
| 广告入口 | 显示即时奖励、目标和观看后差额；全部通用主题已拥有后隐藏 |
| 棋盘模式 | 新存档默认跟随季节；支持每季选择和全局固定棋盘 |
| 旧存档 | 默认原野迁移为跟随季节；主动选择过其他棋盘的存档保留固定模式 |
| Admin 返回 | Admin 试玩链接不再经过玩家语言前缀，避免`/zh/admin/levels` 404 |
| 局内菜单 | 顶部提供菜单；底部只保留局面提示，不再常驻“提示/帮助/更多”三等分栏 |

经济模拟表示“碎片按目标顺序立即兑换”的可达节点，不是玩家实际购买率、广告观看率或收入预测。
