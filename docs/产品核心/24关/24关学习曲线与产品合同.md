# 24关学习曲线与产品合同

> 本文是24关跨关编排的唯一长期来源，回答玩家怎样成长、每关为何存在、羊方主动做什么、玩家怎样反制以及哪些体验绝不能出现。单关细节维护在各关设计卡，通用方法见[关卡与平衡设计标准](../02-关卡与平衡设计标准.md)。

## 一、主线目标

24关不是24组坐标不同的地图，而是24场有顺序的策划遭遇。每一关都必须承接玩家已有能力，提出一个新的主要问题，让羊方用本关特有的计划施压，并提供玩家能够理解、练习和复盘的反制方式。

主线不以“修补现有 24 张地图直到自动门禁通过”为目标，而要建立一套玩家愿意学习和持续挑战的体验：

- 学习顺序清晰；
- 每关有唯一主要问题；
- 相邻关的视觉和策略可以用一句玩家语言区分；
- 春季前段允许宽容，后续羊方不明显送子；
- 高难关给狼留下可理解的取胜空间；
- 岩石服务路线和教学，不承担全部难度；
- 胜率、策略增益、失败归因和新鲜度共同决定放行。

## 二、四季体验职责

| 季节 | 玩家阶段 | 羊方对手表现 | 地图结构目标 |
|---|---|---|---|
| 春 | 学规则、首次捕食、基础协作 | 前两关宽容，后四关逐步减少送子 | 开阔和少石为主，视觉简单 |
| 夏 | 形成策略、管理退路 | 换线、封锁落点、诱导困狼 | 分流、侧翼、双通道和一个代表漏斗 |
| 秋 | 地形计算、连吃与收手 | 熟练利用关键点，但不靠密岩自动封锁 | 少石与密岩混合，只保留少量高密度题 |
| 冬 | 综合考试、开放盘和高压防守 | 接近最强公平防守，主动分散破坏覆盖 | 开放盘为主，允许 1–2 张少石变化，避免六张空盘同质化 |

## 三、24关AI难度曲线

### 1. 科学判断

玩家水平会随关卡提升，但AI难度不能通过“前期故意送、后期突然最优”制造。24关共享同一智力底线：有明确安全替代时不选严格劣着。难度增长来自五个可解释变量：

1. 安全候选从较宽的安全集合逐步收紧到接近最优；
2. 计划从即时反应逐步扩展到完整狼回合和多回合合围；当前目标狼、持续时间与每只狼的移动/捕食历史必须随对局保存和回放；
3. 对手行为从基本自保逐步增加堵落点、围孤狼、地形利用、合理牺牲和反制固定套路；
4. 高难档逐步减少随机并列选择，提高确定性和优势转化；
5. 地图、开局和AI不应在同一关同时大幅跳升，每两关采用“引入—练习”，季末采用综合考试。

每关的`opponentIntent`还必须指定至少两个合法关键区域和普通换目标的最短持续时间。关键区域让“封哪条路”成为可执行评分，目标记忆让“围哪只狼”跨回合持续；二者都必须在正式对局、重开、存档、回放、Admin接管和模拟中保持一致。

### 2. 十二个两关阶段

这些是体验档位，不要求实现十二套独立算法；代码可以由少量稳定能力配置，加上行为风格、本关意图、搜索预算和候选宽度实现。

使用五个生产画像承接十二阶段：`guided`（春01/02）、`foundation`（春03–06）、`tactical`（夏01–06）、`strategic`（秋01–06）、`expert`（冬01–06）。画像是关卡配置，不是玩家可选模式；同一画像内再由地图、开局和每两关教学职责形成细微递进。旧三档算法只用于开发诊断，不属于生产关卡字段。

| 关卡 | 玩家成长阶段 | 羊方目标行为 | 不允许出现 |
|---|---|---|---|
| spring-01/02 | 规则与第一次捕食 | 基础合法、安全候选中保留宽容和变化 | 严格劣着、连续送子、突然合围新手 |
| spring-03/04 | 短连吃与诱导 | 识别立即跳吃，开始堵一次落点 | 玩家尚未学会前就执行多步强封锁 |
| spring-05/06 | 双狼协作与春季综合 | 稳定避吃、限制单狼机动 | 仍靠随机送子让玩家通过 |
| summer-01/02 | 路线和退路 | 利用岩石与通道挡线，预判完整狼回合 | 只保羊、不看狼下一轮连吃 |
| summer-03/04 | 三狼分工与换线 | 主动针对孤立狼，持续一到两步封锁 | 每步更换目标、无意义来回 |
| summer-05/06 | 战术综合 | 识别退路、关键落点和短期牺牲 | 只有档位名称变化但行为没有差异 |
| autumn-01/02 | 窄门与地形控制 | 用关键点压缩入口，形成区域防守 | 仅靠岩石天然封死而AI无决策 |
| autumn-03/04 | 长链风险与跨区 | 评估最大捕食链、跳吃终点和换区 | 只避开第一吃却送出后续长链 |
| autumn-05/06 | 时机与战略综合 | 为困狼接受合理牺牲，优势局主动推进 | 永远保子或只拖到回合耗尽 |
| winter-01/02 | 高压站位 | 多回合维持目标，主动切断狼间接应 | 可避免的明显送子和低级换位 |
| winter-03/04 | 专家防守 | 针对狼方固定套路，合围与反循环并重 | 连续可避免送子、优势下晃动、异常长尾 |
| winter-05/06 | 最强公平综合 | 接近本期生产AI上限，稳定兑现困狼机会 | 隐藏作弊、读取未来操作、靠纯拖延制造困难 |

### 3. 玩家体验目标

- 春季：学习玩家通常1–2次尝试可以理解并通过；失败主要用于教规则和基础策略。
- 夏季：通常1–3次尝试，必须开始组合两狼或三狼，而不是沿用单狼追羊。
- 秋季：通常2–4次尝试，失败应能归因到落点、连吃终点、地形或收手时机。
- 冬季：通常2–5次尝试；资深玩家不应无思考连续一次通过六关，但正确策略最终应有稳定胜路。

以上是上线前校准目标，不是假装已经获得的玩家数据。上线后按到达该季节的玩家分层统计首次通过、三次内通过、重试和流失，再调整目标区间。

### 4. 自动与人工门槛

- 全24关正式生产画像 `chosenDominated=0`。
- 春季可以宽容，但不能用严格劣着制造教学。
- 夏季起必须预判完整狼回合；秋季起必须评估捕食链和地形控制；冬季必须具备多回合目标与优势转化。
- 相邻两关至少在地图问题或羊方对手方式上有清楚差异。
- 自动代理只能证明风险和胜路；季节尝试次数、压力和乐趣必须由真实玩家验证。

## 四、24关完整产品合同

玩家反制是关卡的公平出口；禁止体验是自动化、棋谱和真人共同检查的拒绝线。每关设计卡必须展开本表，而不是只复制一句地图说明。

| 关卡 | 玩家第一次需要做到 | 羊方主动意图 | 玩家反制 | 禁止体验 |
|---|---|---|---|---|
| spring-01 | 移动并完成第一次捕食 | 保持基本队形并占住接近点，但保留教学窗口 | 中狼接触、两翼支援 | 连续送子或突然强围新手 |
| spring-02 | 看懂岩石阻挡并绕行 | 借边石改变接触顺序和落点 | 识别被挡路线后主动换线 | 偏容易却拖成长局 |
| spring-03 | 判断第一次短连吃 | 填补一个短链窗口并从另一侧分散 | 先看连吃终点，再决定继续或收手 | 用直接送子制造连吃教学 |
| spring-04 | 先诱导、再回切捕食 | 把狼引向边线并通过换位拉开距离 | 安全调整三狼间距后从侧面回切 | 羊无目标往返，玩家只能等待 |
| spring-05 | 第一次双狼夹击 | 把主攻狼引向单线并从另一侧转移 | 两狼分压、一狼控制出口 | 两条路线没有真实选择差异 |
| spring-06 | 组合夹击、短链和收手 | 借楔形切断接应并惩罚单狼深入 | 明确三狼分工后再启动短链 | 仍靠随机送子完成季末考试 |
| summer-01 | 集中突破一条封锁线 | 封一线、诱导狼投入另一线 | 集中两狼施压并及时换侧 | 羊只拖延，长局没有空间变化 |
| summer-02 | 捕食前保留第二出口 | 横向封路并把进攻狼推向边角 | 保留第二只狼作为接应和横切 | 难度突然且失败无法归因 |
| summer-03 | 用三狼分工处理漏斗 | 围孤狼、封漏斗落点并切断支援 | 一狼诱开、两狼封口与收割 | 岩石天然封死，AI没有实际决策 |
| summer-04 | 同时维持两侧压力 | 分成两组换线，迫使狼错误调动 | 不把三狼挤到一侧，轮换施压 | 与 summer-01 只是换石版本 |
| summer-05 | 识别边线陷阱和退路 | 借边线与岩石持续压缩一只狼 | 捕食前确认第二出口和接应狼 | 天然死局或无法看懂的突然困狼 |
| summer-06 | 三狼协作后切换主攻侧 | 在两条压力线间换防并拆固定猎手 | 先形成整体压阵，再换侧收网 | 重复利用同一岩石间隙即可获胜 |
| autumn-01 | 读取一个关键斜门 | 占住斜门出口并分离第三只狼 | 两狼控门、一狼沿可退出路线推进 | 地形自动完成封锁 |
| autumn-02 | 控制主通道两端 | 一端封锁、一端分散，破坏双端控制 | 两狼守两端，第三狼等待窗口 | 只有一条死背答案 |
| autumn-03 | 权衡长连吃和主动收手 | 改变连吃落点，诱导狼为多吃失去退路 | 先算最终落点，再决定链长 | 羊不可避免排队或地图过度偏狼 |
| autumn-04 | 跨区前计算下一落点 | 在双岛间切断接触并迫使提前选区 | 控制两端后再跨区 | 永久分区、无互动长局 |
| autumn-05 | 识别短暂时机窗口 | 在窗口形成前换防并关闭另一端 | 同时保持两端通行，等待强制机会 | 关键窗口无法从盘面读出 |
| autumn-06 | 综合换区、长链和机动 | 分散到多岛并持续切断连收路线 | 先稳住一区，再转移出口控制 | 只靠岩石密度制造季末难度 |
| winter-01 | 无地形支点仍保持三狼覆盖 | 选定低机动狼并逐步压缩出口 | 宽覆盖、轮换接触，不追单羊 | 专家羊仍只保子或无故留洞 |
| winter-02 | 在单锚点下切断双翼回环 | 围绕锚点分散并切断狼间接应 | 稳住中心后从边线建立首吃 | 长尾主要来自羊往返 |
| winter-03 | 高压空盘中等待决定性机会 | 持续针对最弱狼并破坏三狼对齐 | 保持覆盖，等薄弱侧出现再兑现 | 连续送子、每回合更换目标 |
| winter-04 | 不追单羊并轮换三狼职责 | 绕石回环诱导追逐，同时收紧另一侧 | 维持全盘控制并主动更换进攻狼 | 重复局面成为羊方主要策略 |
| winter-05 | 先合围，再撕开边线 | 借双柱拉散阵型并孤立固定猎手 | 维持两翼线后由边狼突破 | 同一狼反复得分而羊不反制 |
| winter-06 | 在一局中组合四季能力 | 识别玩家主猎手，双门换防并主动完成困狼 | 每轮捕食后重建分工和退路 | 纯拖延、重复收割或与 winter-05 同解 |

## 五、快速生产与淘汰流程

每个重做关先生成 3 个明显不同的候选，不做十几个坐标微调。候选先过快速门禁：

- 结构合法；
- 与现有地图家族不重复；
- random、mixed、chain-aware 各 5 个固定种子；
- 线上同档羊 AI；
- 首次捕食、胜负、平局、连吃敏感和代表棋谱；
- 玩家语言可以说明该关与前后关的差异。

明显重复、无首吃、羊明显送子、稳定机械强胜或失败无法解释的候选立即淘汰。每关只保留 1–2 个候选进入更多种子、有限求解和人工试玩。

## 六、当前生产边界与缺口

- 24张地图均已完成结构重建、地图指纹、新鲜度和生产配置接入；非空岩石布局不存在相同、镜像或旋转等价组合。
- 最新自动状态和逐关人工入口只在[24关总览](README.md)维护；当前固定种子证据见验证记录中的羊 AI V5 报告。
- 十二个“引入—练习”阶段和24关三维对手合同已经接入代码；Admin可显示能力、风格、本关意图、目标狼、换目标原因、历史捕食、关键区域和行动影响。自动门禁已覆盖主动压力、目标持续、受控交换、反固定猎手、无进展、风格差异和独立教师后悔值。
- 自动结果用于发现结构、长尾、首次接触和AI动作风险，不等于玩家体验通过。
- 关卡调整必须重新检查学习职责、新鲜度、羊AI画像、固定种子对抗和玩家反馈，不能只追求某个胜率数字。
- 当前V5是首发主动对手候选，不是完美求解或真人替身。自动侧收口后，冬季专家感、计划是否能被玩家读懂和整套关卡的乐趣，只通过有限定向体验和上线数据继续判断。
