>
> 当前总判断：24 个关卡均可运行并具备设计档案。修复模拟反重复、随机流耦合和 Hard 有限视野后，当前为 11 关 pass、13 关 review、0 关 reject；review 主要表示策略收益需要玩家验证。

## 状态定义

- `pass`：未命中当前自动门禁红线，可进入玩家试玩，但不代表已验收。
- `review`：命中和棋、长尾或策略敏感风险，必须完成棋谱复盘并决定修订或人工放行。
- `reject`：命中明确拒绝规则，当前不得作为合格生产关卡，必须修订并复验。
- `用户试玩确认`：用户完成实际操作验证，确认难度、策略可理解性和体验符合预期。

### “策略敏感”如何解释

- 它表示诊断用 `mixed` 狼策略的胜率显著高于 `random`，证明“认真规划比随意走更有效”，这是策略游戏需要的性质。
- 它同时是待验证风险：只有当玩家能够从盘面、关卡文案和失败反馈中发现并执行该策略时，差距才代表合理难度；若只能查看标准答案或依赖羊 AI 失误，地图仍不合格。
- 因此 13 张 review 关卡均已在单关设计卡维护推荐策略、盘面信号、常见错误、失败学习和玩家试玩问题。`spring-03`、`winter-05` 还必须复盘长尾局。
