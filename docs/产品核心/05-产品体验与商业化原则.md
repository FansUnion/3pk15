# Fangrush 产品体验与商业化原则

> 本文维护游戏核心、玩家体验和商业化之间不可违反的长期原则，不维护渠道进度、平台字段、模拟数字或实现细节。渠道路线见[渠道与变现路线](../商业成功/02-渠道与变现路线.md)，未来内容见[产品扩展](../产品扩展/README.md)。

## 一、产品目标

Fangrush首先必须是一款玩家愿意学习、失败后愿意重试、获胜后愿意继续的策略游戏。商业化只能建立在这种价值上，不能替代它。

判断优先级：

1. 玩家能理解并信任规则；
2. 24关具有成长、新鲜感和可执行胜路；
3. AI提供公平、可读的防守压力；
4. 操作、视听和奖励让结果有感知价值；
5. 广告、IAP和长期内容在不伤害以上体验的前提下获得收入。

## 二、玩家不负责解释系统

- 页面和局面必须用玩家语言说明发生了什么、为什么结束、下一步能做什么。
- 玩家只需反馈“看不懂、太笨、过难、拖延、听不清、不好看”；开发侧负责复现、归因和技术解释。
- 自动化术语、函数名、评分权重和内部状态不能作为玩家理解前提。
- 失败必须能归因到策略、地图、AI或明确规则，不能表现成随机惩罚或系统错误。

## 三、通关与继续游戏

终局信息顺序应当是：

```text
结果与原因
  -> 获得的基础奖励
  -> 下一关或重试
  -> 分享、皮肤或可选激励
```

下一关或重试始终是主要操作。分享和广告不能比继续游戏更突出，也不能让玩家误以为必须先操作商业按钮。

## 四、广告原则

### 广告不是玩家的任务

- 不看广告也能完成主线、领取基础奖励和继续游戏。
- 首次进入、教学、选择、移动、捕食、连吃和羊AI行动中不展示广告。
- 广告失败、取消、无填充或被拦截时，输入、声音、存档和基础奖励必须可靠恢复。
- 不用广告解决坏地图、AI低级错误或失败解释不足。

### 广告必须有可见价值

玩家不会为了项目收入看广告，只会为了当前可理解的目标。激励需要同时满足：

- 自愿；
- 播放前说明获得什么；
- 成功后立即、幂等兑现；
- 能推进皮肤或近期收藏目标；
- 不提供战力、必胜提示或隐藏难度优势。

商业按钮处于次要层级不等于没人点击。真正影响点击的是奖励是否清楚、有用、及时，以及玩家是否已经产生继续游玩的意愿。

## 五、免费核心与付费边界

必须免费、完整和公平：

- 核心规则与四季24关主线；
- 基础帮助、失败解释和正常提示；
- 本地进度、基础奖励和至少一套清楚可用的正式外观；
- 相同关卡中的规则、AI能力和胜负条件。

可以作为长期商业内容：

- 不影响胜负的高质量皮肤和主题棋盘；
- 去广告；
- 独立关卡包或公开规则的高级挑战；
- 有真实玩家需求后建立的收藏和展示能力。

禁止出售：

- 更高连吃次数、削弱羊AI、额外狼或必胜走法；
- 主线必需帮助、正常重试或基础奖励；
- 暗中按付费、广告观看或失败次数操纵难度的能力。

## 六、奖励与皮肤

- 玩家先知道本局获得什么、当前拥有多少、距离目标还差多少。
- 皮肤价值来自轮廓、主题、材质和实装效果，不来自价格或稀缺标签。
- 图鉴、装备状态和棋盘实际表现必须一致。
- 碎片和任务提供近期目标，但不能强迫玩家重复刷低价值关卡。
- 每季主题可以作为主线成长奖励；更多付费或高级主题只有达到审美和收藏标准后才加入。

具体奖励、经济里程碑和广告产品合同见[奖励、经济与商业化设计](04-奖励经济与商业化设计.md)。

## 七、AI与难度

- 前期宽容来自安全候选和教学节奏，不来自故意送子。
- 后期压力来自完整捕食链预判、合围、地形控制和优势转化，不来自作弊。
- 主线允许高水平玩家击败AI，但胜利应像拆开防线，不像羊排队送子。
- 自动模拟负责发现风险和保证动作底线；玩家试玩负责判断是否聪明、有趣、过难或拖延。
- 更强AI、残局证明和大师挑战属于长期内容，不能反向提高首发主线门槛。

具体AI能力和验收见[羊AI机制与制作人验收指南](09-羊AI机制与制作人验收指南.md)。

## 八、跨平台一致性

- 独立站、H5、Admin和未来App共享同一规则、AI和24关配置。
- 平台只改变生命周期、广告、分享、存储、支付和外部能力。
- 不支持的能力应隐藏或友好降级，不能留下失效按钮。
- 广告和支付回调不能直接修改核心规则状态。
- 任一渠道的基础对局、奖励和恢复不依赖广告填充或账号成功。

## 九、用户反馈处理

每个问题归入规则、地图/开局、羊AI、教学/文案、交互、视听资源、奖励或平台能力之一。

- 单局AI或终局问题使用可复现问题包和棋谱；
- 难度、新鲜感、审美和声音使用真实设备与玩家描述；
- 修复后同时检查直接问题、相邻关卡、存档恢复和跨平台边界；
- 单次体感和单次模拟都不能独立推翻长期标准，但可触发有证据的复核。

## 十、商业健康判断

商业结果必须与体验指标一起解释：首局完成、失败后重试、下一关点击、每次会话局数、回访、皮肤目标理解、广告完成及广告后继续。

更高广告次数、更低玩家胜率或更长游戏时长，如果同时降低重试、继续和回访，就不是成功。具体指标与阶段决策见[商业定位与成功指标](../商业成功/01-商业定位与成功指标.md)。
