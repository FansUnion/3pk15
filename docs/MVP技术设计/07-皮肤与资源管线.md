# 07 · 皮肤与资源管线

> 玩法侧约定见 [`docs/游戏创意/07-内容进度-碎片皮肤任务.md`](../游戏创意/07-内容进度-碎片皮肤任务.md)。

---

## 1. 概念

| kind | 说明 |
|------|------|
| `wolf_set` | 狼外观 + **成套羊外观**（强制成对） |
| `board` | 棋盘主题（线色/背景）；与狼皮独立 |
| 岩石 | **无皮肤**，全局默认一式 |

存档只存皮肤 **id**；资源在包内。

---

## 2. SkinCatalog 契约

```ts
type SkinCatalogItem =
  | {
      id: string
      kind: 'wolf_set'
      name: string
      unlock:
        | { type: 'default' }
        | { type: 'chapter'; chapterId: ChapterId }
        | { type: 'cost'; universal: number }
      assets: {
        wolf: string      // 相对 content 路径，如 skins/frost/wolf.svg
        sheep: string
        thumb?: string
      }
    }
  | {
      id: string
      kind: 'board'
      name: string
      unlock: { type: 'default' } | { type: 'chapter'; chapterId: ChapterId } | { type: 'cost'; season: ChapterId; amount: number }
      assets: {
        boardBg: string
        thumb?: string
      }
    }
```

### 校验（CI 或 `pnpm check:skins`）

1. 每个 `wolf_set` 的 `wolf`/`sheep` 文件存在。  
2. id 唯一。  
3. 至少 1 个 default wolf_set + 1 个 default board。  
4. 成本解锁的数值 ≥ 0。

缺羊资源 → **失败**，禁止合并。

## 3. 目录规范

```
content/skins/
  default/
    wolf.svg
    sheep.svg
    thumb.svg
  frost/
    wolf.svg
    sheep.svg
  boards/
    default.svg
    spring.svg
    summer.svg
    autumn.svg
    winter.svg
```

- 优先 **SVG**；viewBox 统一（建议棋子 `0 0 64 64`）。  
- 视觉：牧场浅色；剪影 1 秒可辨；避免紫霓虹。  
- 三态若需要：`wolf.svg` 内用 CSS class 或单独 `wolf-selected.svg`（MVP 可用描边在渲染层加，不强制多文件）。
