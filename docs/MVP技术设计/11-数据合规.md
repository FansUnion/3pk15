# 11 · 数据合规与隐私（一期）

## 1. 原则

- **无账号**：不收集邮箱、手机、实名。  
- **对局逻辑本地**：棋谱、AI 不上传自建服务器。  
- **门户默认零自建远程遥测**；独立站统计可后接且须可关。

---

## 2. 本地数据清单

| 数据 | 存储 | 敏感性 |
|------|------|--------|
| SaveGame（进度、碎片、皮肤 id、引导、buff） | localStorage | 低；仅设备本地 |
| 音量等设置 | localStorage | 低 |
| Admin 密钥会话 | sessionStorage | 中；勿持久 |

**不存**：身份证、支付信息、精确通讯录、相册。

---

## 3. 第三方

| 方 | 数据 | 要求 |
|----|------|------|
| 广告 SDK | 按厂商隐私政策；可能有广告 ID / cookie | 结算/激励前可先出简短说明链接；失败跳过 |
| 门户平台 | 受门户条款约束 | 不另加自建分析像素（除非门户要求） |
| Vercel 托管 | 标准访问日志（IP 等由托管方处理） | 独立站隐私页说明「托管商日志」即可 |

---

## 4. 独立站页面（建议文案要点）

提供 `/privacy`（可极简）：

1. 游戏进度存在浏览器本地。  
2. 看广告时适用广告商政策。  
3. 无账号则无法跨设备同步。  
4. 联系邮箱（个人开发者）。

设置页链到该页。

---

## 5. 遥测（若独立站后接）

仅允许：

- 匿名事件：`level_clear` / `level_fail` / `ad_reward_ok`（无棋盘 FEN、无个人标识）。  
- 一键关闭或构建开关 `TELEMETRY=0`。  
- **门户包默认关闭。**

禁止：上传完整棋谱、皮肤资源中的用户图（无 UGC）。

---

## 6. 儿童与分区

- 定位益智棋类；广告需选相对合规网络。  
- 若上架地区有 COPPA/GDPR 硬性：二期再评估年龄门；一期文档标明风险，上架前人工确认目标商店条款。

---

## 7. 与后台

Admin 模拟局 **默认不上传**；导出 JSON 仅本机下载。
