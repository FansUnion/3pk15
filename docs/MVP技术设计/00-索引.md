# MVP 技术设计 · 索引

## 第一期 MVP 定义

**第一期 MVP =《游戏开发计划》阶段 1（核心可玩）+ 阶段 2（留存与商业化）**，并以 **独立站优先上线**。

目标：

1. **独立站**（Vercel）可玩、可留存、可广告；含 **简易后台**（皮肤/关卡只读、AI 模拟、玩法说明）。  
2. 可打 **门户 H5 瘦包**（无 `/admin`、默认零自建遥测）。  
3. **不含**：玩家账号/云存档、付费商城、多语言、App 上架。

| 做 | 不做 |
|----|------|
| 6×6 对局、连吃≤5、三档 AI、四季关卡、本地进度 | 账号 / 云存档 |
| 碎片、皮肤图鉴与穿戴、日/周任务 | 付费去广告、商城 |
| 插屏 + 激励（补碎片 / 双倍 30 分钟） | 多语言、分享截图、每日挑战、星级、撤销 |
| 独立站 `/admin` 配置预览与 AI 模拟台 | 运营报表后台、在线热改关卡入库 |
| TypeScript + Next + SVG + 客户端 AI | Phaser/Unity；AI 放服务端 |
