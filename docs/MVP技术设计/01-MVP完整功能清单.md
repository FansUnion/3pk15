# 01 · MVP 完整功能清单

范围：第一期 MVP（阶段 1 + 阶段 2）。  
格式：`ID | 名称 | 验收标准 | 规则依据`

规则依据指向 `docs/游戏创意/` 下编号文档。

## A. 对局核心

| ID | 名称 | 验收标准 | 规则依据 |
|----|------|----------|----------|
| A01 | 6×6 交点棋盘 | 渲染 6×6 交点与横竖连线；棋子落在交点；PC/手机正方形等比不拉伸 | 02、03、08 |
| A02 | 坐标系 | 全局 `(行1–6, 列1–6)`；开局狼/羊坐标与 03 一致 | 03 |
| A03 | 图层遮挡 | 显示顺序：岩石 > 棋子 > 棋盘线 | 08 |
| A04 | 开局生成 | 3 狼 + 15 羊固定点；可按关卡叠岩石 0/1/2/4/6/8 | 03、04 |
| A05 | 仅操作狼 | 点击可选中狼；羊、岩石不可作为操作目标 | 01、08 |
| A06 | 合法走位提示 | 选中狼后，横竖相邻空点高亮（可走） | 02、08 |
| A07 | 走一格 | 狼移到相邻空点；不可斜走；不可穿岩石/棋子 | 02 |
| A08 | 隔空吃 | 同线「狼—空—羊」可移至羊位并移除羊；紧贴/中间有子不可吃；非法不执行 | 02 |
| A09 | 吃子路径提示 | 可吃时中间空与羊位高亮（与可走区分，建议红/强调色） | 08 |
| A10 | 连续多吃 | 同狼同回合可连吃；**最多 5 次**；满 5 或无法再吃则结束该狼回合；连吃中保持选中 | 02 |
| A11 | 羊移动规则约束 | AI 羊仅前进/横走，禁止向第 1 行后退；无吃子 | 02 |
| A12 | 阻挡判定 | 岩石与棋子阻断走子与隔空吃中间空 | 02、04 |
| A13 | 胜利 | 累计吃羊 ≥8 立即胜；弹出结算 | 02 |
| A14 | 失败 | 三狼皆无合法走/吃立即败；弹出结算 | 02 |
| A15 | 无和棋 | 仅胜/败两种终局 | 02 |
| A16 | 回合切换 | 狼先手；玩家完成本回合（走或连吃结束）后进入 AI 羊回合 | 02 |
| A17 | 重置本局 | 恢复本关开局局面与吃羊计数 | 08 |
| A18 | 退出对局 | 返回关卡列表；进度按通关规则已存则保留 | 08 |

## 附录：一期明确不做

以下出自源稿二期/三期，**不进入 MVP 验收**：

- 账号登录、云端存档、跨设备同步  
- 去广告付费、商城、付费皮肤直购  
- 多语言、分享对局截图  
- 每日挑战关、星级评分  
- 操作撤销（广告解锁）  
- 棋子/连吃华丽特效、四季主题重纹理动画（可用静态换色/换皮满足 D）  
- 玩家侧 AI 难度滑块  
- 运营向「AI 行为日志 / 广告曝光报表」持久化后台（**不等于** H05 模拟台）  
- 后台在线编辑关卡并写数据库  

---

## 验收总门槛（建议）

1. 春→夏→秋→冬各至少 1 关可完整胜/负，无规则死锁  
2. 清缓存后进度丢失；有缓存时刷新可恢复  
3. 关广告 SDK 模拟失败时，结算与激励链路仍可继续玩  
4. 独立站：H05 能用 hard 档复现至少 1 个合围类 fixture 局面
