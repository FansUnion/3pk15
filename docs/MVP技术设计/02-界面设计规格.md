# 02 · 界面设计规格

移动优先；PC 居中同构（最大宽度约 480–560px 内容柱）。  
视觉：草地牧场、四季浅色氛围；**避免**紫霓虹赛博风与厚重拟物阴影堆叠。狼/羊剪影须 1 秒可辨；岩石为简单灰石块。

规则细节见 `docs/游戏创意/`；本页只规定呈现与交互。

---

## 1. 主流参考（模式借鉴，不抄素材）

| 类型 | 借鉴点 | 落到本品 |
|------|--------|----------|
| H5 棋盘休闲（Poki/CrazyGames 上 Chess / Checkers / Fox & Geese 类） | 棋盘绝对主视觉；HUD 一条顶栏；少侧栏 | 对局页棋盘 ≥70% 可视高度 |
| 闯关休闲（章节岛 + 关卡格） | 章节大门 → 关卡网；锁/星/勾 | 四季章节 → 关卡列表锁/通关 |
| 收集轻壳（皮肤/碎片类） | 结算强 CTA；图鉴网格；「看广告得币」次级按钮 | 结算「再来一局」主按钮；激励次按钮 |
| 益智引导 | 首次半透明遮罩 + 高亮可点区 | 仅春日第 1 关 |

---

## 2. 全局导航

```
首页
 ├─ 开始 / 继续 → 章节选择 → 关卡列表 → 对局 → 结算 →（再来 | 回关卡 | 回首页）
 ├─ 图鉴（皮肤）
 ├─ 任务
 └─ 设置
```

- 对局中底部仅保留：**重置 | 静音 | 退出**（退出回关卡列表）  
- 图鉴/任务/设置不对局内嵌套过深；结算后可快捷进图鉴（可选入口）

## 3. 页面规格

### P1 首页

```
+----------------------------------+
|          狼与绵羊                 |
|        Wolf & Sheep              |
|                                  |
|     [  开始冒险 / 继续  ]         |
|                                  |
|  [图鉴]   [任务]   [设置]         |
+----------------------------------+
```

| 项 | 规格 |
|----|------|
| 必显 | 标题、主 CTA、图鉴/任务/设置 |
| 主 CTA | 无进度时「开始冒险」→ 春日；有进度时「继续」→ 最近未通或上次章节 |
| 空态 | 无 |

---

### P2 章节选择

```
+----------------------------------+
|  ← 返回                          |
|     选择季节                     |
|  [春日] [夏日] [秋日] [冬日]      |
|   开    锁?   锁?   锁?          |
+----------------------------------+
```

| 项 | 规格 |
|----|------|
| 必显 | 四季卡片；锁定态灰显+锁标；已解锁可点 |
| 锁态 | 未通上一章全部关 → 不可进；可展示「通关夏日解锁」类短句 |
| 视觉 | 四季风味色微差（春嫩绿/夏草绿/秋暖黄/冬冷青），仍属浅色牧场系 |

---

### P3 关卡列表

```
+----------------------------------+
|  ← 春日                 碎片 n   |
|  [1 通] [2 通] [3 ·] [4 锁] ...  |
+----------------------------------+
```

| 项 | 规格 |
|----|------|
| 必显 | 关卡序号网格；通关勾；锁定；当前可玩高亮 |
| 交互 | 点可玩关进入对局 |
| 角标 | 右上可显通用碎片余额（可选，增强收集感） |

---

### P4 对局主界面（核心）

```
+----------------------------------+
| 春日·第1关          已吃 2/8  羊13|
+----------------------------------+
|                                  |
|         【正方形棋盘】            |
|                                  |
+----------------------------------+
|  [重置]      [静音]      [退出]   |
+----------------------------------+
```

| 项 | 规格 |
|----|------|
| 顶栏左 | 季节 · 关卡名 |
| 顶栏右 | **已吃 x/8**（胜利目标）、剩余羊数 |
| 棋盘 | 居中正方形；交点可点；线不抢视觉 |
| 选中狼 | 明确描边/提高对比 |
| 可走 | 绿色半透明点（或软绿圈） |
| 可吃 | 红色/暖色落点；可与路径短线提示 |
| 连吃中 | 保持该狼选中；顶栏可短暂显示「连吃 2/5」 |
| AI 回合 | 简洁「绵羊行动中…」防误点（半透明遮罩可选） |
| 底栏 | 重置（需二次确认）、静音、退出 |

**遮挡**：岩石 > 棋子 > 线。

---

### P5 结算弹窗

```
+----------------------------------+
|         胜利 / 失败              |
|      获得碎片 +n  （种类）        |
|                                  |
|     [ 再来一局 ]     ← 主按钮     |
|     [ 返回关卡 ]                 |
|     [ 看广告双倍碎片 ] ← 次要     |
+----------------------------------+
      （插屏位：弹窗流程前后）
```

| 项 | 规格 |
|----|------|
| 胜 | 强调已吃 ≥8；展示掉落碎片 |
| 负 | 短文案鼓励重试；仍可发任务相关提示（若有） |
| 主 CTA | 「再来一局」重开本关 |
| 插屏 | 胜/负结算流程中触发（见功能 F01）；失败加载则跳过 |
| 双倍 | 若未处于双倍 buff 中，提供激励入口 |

---

### P6 皮肤图鉴 / 穿戴

```
+----------------------------------+
|  ← 图鉴          通用碎片 n      |
|  [狼] [棋盘]  Tab               |
|  [皮1 穿戴中] [皮2 锁 40碎片]   |
|  ...                             |
|  [ 使用 ]  /  [ 碎片不足·看广告] |
+----------------------------------+
```

| 项 | 规格 |
|----|------|
| Tab | 狼皮（含说明羊会成套变）、棋盘皮 |
| 卡片 | 预览、已解锁/未解锁、价格（通用或四季碎片） |
| 羊皮 | 不单独列表；文案说明「跟随狼皮」 |
| 岩石 | 不出现在图鉴 |

---

### P7 任务

```
+----------------------------------+
|  ← 任务                          |
|  每日                            |
|  [ ] 通关 1 次    +碎片  [领取]  |
|  每周                            |
|  [x] 吃羊累计 20  [已领]         |
+----------------------------------+
```

| 项 | 规格 |
|----|------|
| 分组 | 每日 / 每周 |
| 态 | 未完成 / 可领取 / 已领取 |
| 领取 | 灰态不可点；可领主色按钮 |

---

### P8 设置 / 玩法说明

```
+----------------------------------+
|  ← 设置                          |
|  音效    [开关]                  |
|  玩法说明 [查看]                 |
+----------------------------------+
```

玩法说明弹层：短条文 — 狼先手、隔空吃、连吃最多 5、吃满 8 胜、三狼无步败、羊不能后退。

---

### P9 新手引导（春日第 1 关）

| 步 | 内容 |
|----|------|
| 1 | 高亮一只狼：「点选你的狼」 |
| 2 | 高亮可走点：「走到闪烁的位置」 |
| 3 | 若有教学吃子局面或图示：「隔一空点羊就能吃掉」 |
| 4 | 「一回合最多连吃 5 次；吃够 8 只羊过关」→ 关闭 |

- 仅该关首次（本地记 `guide_v1_done`）  
- 可跳过  

---

### P10 激励相关浮层

| 场景 | UI |
|------|----|
| 合成皮碎片不足 | 模态：还差 n →「看广告领取」/「取消」 |
| 双倍 | 关卡列表或结算：「开启 30 分钟双倍」；生效中显示倒计时条 |
| 广告失败 | Toast「暂时无法播放，稍后再试」后关闭，不挡操作 |

---

## 4. 关键状态汇总

| 状态 | 表现 |
|------|------|
| 章节锁 | 灰 + 锁 |
| 关卡锁 | 灰 + 锁 |
| 关卡通关 | 勾或草地徽记 |
| AI 思考 | 短文案，禁止点棋盘 |
| 连吃中 | 「连吃 k/5」 |
| 双倍 buff | 顶角或列表角倒计时 |
| 广告中 | 全屏交给广告 SDK；返回后恢复原页 |

---

## 5. 响应式

- **竖屏手机**：对局底栏固定；棋盘宽度 = 屏宽 − 左右边距  
- **横屏/PC**：内容柱居中；棋盘边长取 min(柱宽, 可视高×0.72)  
- 禁止棋盘非等比拉伸
