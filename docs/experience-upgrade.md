# Fangrush 体验升级任务

## 目标

把官网、游戏内界面和游戏素材统一成一个有辨识度的羊狼棋盘游戏体验。玩家应在第一次进入时看懂目标，在每一步操作时看懂结果，并愿意继续挑战。

## 优先级

1. 棋盘视觉层级、羊/狼角色表现和状态动画。
2. 走子、跳吃、连锁、胜利、失败的反馈与音效同步。
3. 官网首页主视觉，以及皮肤、关卡、玩法、设置页面的统一风格。
4. 角色皮肤、棋盘主题、环境音和可扩展资源结构。
5. 玩家体验验证：首次理解、首次操作、首次失败、胜利后继续游玩。

## 资源方向

- 羊：默认、受惊、移动、被围困、逃脱、失败和庆祝状态；皮肤应改变主题气质，不只换颜色。
- 狼：待机、追踪、扑击、吃子、被阻挡、胜利和失败状态；动作节奏比羊更快、更有压迫感。
- 棋盘：格子、环境、交互提示、动态反馈四层；春夏秋冬主题要包含背景、格子、边框、角色适配和环境音。
- 音效：操作音、规则反馈音、情绪音、环境音四层；普通操作轻，连锁和胜负有明显层次。

## 验收标准

- 3 秒内能看懂这是羊狼棋盘游戏。
- 10 秒内能找到开始游戏并理解下一步操作。
- 棋盘上可走位置、当前角色、危险位置和行动结果不依赖文字解释。
- 普通走子、跳吃、连锁、胜利和失败拥有不同视觉与声音反馈。
- 官网和游戏内页面在颜色、字体、按钮、面板和素材风格上保持一致。
- 减少动画设置开启时，所有核心操作仍然清晰且不阻塞。
