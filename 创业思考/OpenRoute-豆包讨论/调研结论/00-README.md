## 当前已拍板假设（摘要）

1. **产品形态**：独立英文海外 SaaS（纯 BYOK），完全独立品牌与站点，不嵌入 AEO。
2. **首发用户**：海外个人用户 + 隐私敏感 / 开发者。
3. **上游**：OpenRouter 或同类「充值一次、全模型」聚合平台；非逐家官方 Key。
4. **底座（待最终二选一）**：  
   - **路径 V**：LobeChat + Vercel（低成本优先）  
   - **路径 B**：LibreChat + VPS（功能最全）  
5. **架构倾向**：Key 服务端加密 + 后端转发。  
6. **变现**：免费（无广告）+ 2～3 档订阅；官网含 OpenRouter 教程深链。

详情以 [`01-需求说明书.md`](./01-需求说明书.md) 为准。
