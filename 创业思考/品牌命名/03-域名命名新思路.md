# 03 · 域名命名新思路（第二轮）

> 第一轮结论：语义贴切的短造词 .com **99/100 已占用**，唯一可注册的 `kyvent.com` 缺乏信任感。
> 本轮策略转变：**名字空、文案满**——品牌名不必复述产品主旨，tagline 负责说清楚。

---

## 一、策略转变

### 从「语义贴切」到「空壳品牌」

| 旧思路（第一轮） | 新思路（第二轮） |
|------------------|------------------|
| 名字要说清 AI 可见度 / 排名 / 引用 | 名字只说「可信、好记」 |
| 造词优先，≤10 字母 | **.com 可得 + 好读** 是第一优先级 |
| visible / cite / rank + -ora/-io/-ly | 两词组合 / 地名感造词 / 中性实词变形 |
| 语义相关是必选 | 语义相关是加分项，非必选 |

成熟 SaaS 的惯例：

| 品牌 | 名字与产品关系 | 语义从哪来 |
|------|----------------|------------|
| Stripe | 几乎无关支付 | 产品 + tagline |
| Linear | 通用形容词 | 品类文案 |
| Basecamp | 户外隐喻，非功能直述 | 品牌故事 |
| Otterly | 动物 + 后缀 | 品类营销 |
| Peec | 纯造词 | 同上 |

我们的 tagline 已经很强（见 [02-产品定位](../geo/02-产品定位.md)）：

> **Get your brand recommended by AI.**

因此首页可以是：

```
[沉稳品牌名].com
Get your brand recommended by AI.
Track & improve how ChatGPT, Perplexity and Google AI answer about you.
```

---

## 二、品牌气质（本轮锁定）

**沉稳可信** — Linear / Basecamp 感：

- 不卖萌（排除 Otterly 式过度活泼）
- 不企业腔过重（排除 Profound / Athena 式）
- 适合 indie SaaS、$15–49/月 PLG 自助产品

---

## 三、三条新路线

### 路线 1：两词组合（Compound .com）

两个常见英文词拼成一个域名，像地名 / 机构名。

| 模式 | 示例方向 | 为什么 |
|------|----------|--------|
| 自然 + 自然 | CedarHarbor, MapleBrook, StonePine | 稳重、不假科技 |
| 方向 + 自然 | NorthField, StillWater, EastHaven | Basecamp 式信任感 |
| 质感 + 物 | SilverGate, CopperLane, GrayStone | 商务友好 |
| 动物 + 地貌 | HeronPoint, FinchBay, LynxHill | 好记但不卖萌 |

**避开**：`getxxx` / `xxxAI` / `xxxrank` / `BrandTrack` 等功能直述；总长 > 18 字母。

### 路线 2：更长但好读的造词（11–14 字母）

用真实英语音素拼合，听起来像地名或机构名。

| 模式 | 示例方向 |
|------|----------|
| 地名感 | Westmere, Brookhaven, Fairwick |
| 柔和 portmanteau | Clearwick, Steadwell, Truespan |
| 机构感变体 | Clarion, Meridian 类变形 |

**硬过滤**（吸收第一轮教训）：

- 禁止 `kyvent` / `qlyvio` 式随机辅音簇
- 禁止 x/q/z 开头或难拼中间字母
- 必须能通过「读一遍 → 对方能拼」电话测试

### 路线 3：中性品牌实词 / 空壳词

选有好感的英文词，或动物 / 自然词 + 轻后缀，完全不提产品主旨。

- 原词 .com 大概率已占 → 用变形：`Finchwell`, `Heronly`, `Steadio`
- 气质：稳、正面、不功能化

---

## 四、新筛选标准

| 等级 | 标准 |
|------|------|
| **S** | .com 可注册 + 电话拼写测试通过 + 沉稳可信 + 无显著知名品牌冲突 |
| **A** | .com 可注册 + 好读，但略长（12–14 字母）或气质稍弱 |
| **B** | 可注册但偏活泼 / 普通 / 略难拼 |
| **淘汰** | 怪拼、功能直述两词、蹭竞品 |

**不变硬约束**：只考虑 .com；入围后仍需商标 + 社媒 handle + Google 负面检索（见 [06 §五](../geo/06-域名与品牌命名.md)）。

## 六、诚实预期

- 常见两词组合（如 ClearLake）仍可能大量已占；本轮偏向略冷门但好读的组合
- 更长好读造词的可注册率应**高于**第一轮语义造词
- 最终体验：名字负责「可信 + 好记」，tagline 负责「是什么」
