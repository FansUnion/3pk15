# Poki 发行清单

> 最近核对：2026-07-17。执行时以 [Poki Requirements](https://sdk.poki.com/new-requirements.html) 和 [HTML5 SDK](https://sdk.poki.com/html5.html) 为准。

## 集成

- [ ] 初始化Poki SDK；失败时游戏仍可启动。
- [ ] 加载完成、首次输入、暂停、菜单、关卡结束和恢复正确发送生命周期事件。
- [ ] 自然继续节点调用商业广告机会；不自行维护广告频率计时器。
- [ ] 激励广告提前明确奖励，只有成功播放才发放；普通继续选项始终存在。
- [ ] 广告期间暂停输入和音频，结束或失败后可靠恢复。
- [ ] 支持平台分享链接；保存数据满足平台云同步边界。
- [ ] 移除外部广告、无关外链、GA和未经批准的外部请求。

## 游戏与技术

- [ ] 桌面、手机和平板可玩；平台画布按16:9稳定缩放。
- [ ] 隐私模式/localStorage不可用时仍可开始游戏。
- [ ] 首次输入才标记正式Gameplay Start；事件不连续重复。
- [ ] 平台包不包含Admin、调试工具和独立站广告代码。
- [ ] 初始下载尽量控制在官方建议范围内，并通过Poki Inspector。

## 素材

- [ ] 静态方形缩略图：满版、至少628x628、无边框，突出默认狼与真实玩法。
- [ ] 动画缩略图：用于Soft Release至全球发布阶段。
- [ ] 英文名称、描述、操作说明和必要本地化。
- [ ] 缩略图在小尺寸下仍清晰，不使用误导性画面。

## 当前缺口

真实SDK适配器、生命周期事件、广告暂停恢复、平台分享链接、静态/动画缩略图和Inspector验证均未完成。
