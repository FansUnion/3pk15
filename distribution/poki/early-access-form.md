# Poki 早期准入申请

> 表单：[Request early access](https://developers.poki.com/share)。这是申请开发者后台准入，不是上传游戏、签合同或正式上架。联系人、邮箱和国家必须使用用户真实信息。

## Fangrush 填写口径

| 字段 | 建议填写 |
|---|---|
| Name | 用户本人英文名或拼音全名 |
| Email | 长期接收平台和合同邮件的真实邮箱 |
| Studio / team | `Fangrush Studio` |
| Country | 实际常驻国家，例如 `China` |
| Team description | Indie / Solo developer / Small indie studio；首次开发者如实选择 |
| Previous releases | 首次发行则选 None / Not yet，不虚构平台履历 |
| Game links | `https://fangrush.com` |
| Genres | Strategy, Puzzle, Board game, HTML5 casual |
| Engine | HTML5 / TypeScript / Next.js / Custom；不要填写 Unity |
| Expected value | Distribution, reach global players, ads revenue share, publishing support |

作品链接说明可直接使用：

```text
First web game release. Playable demo of Fangrush, an HTML5 strategy board game with 3 wolves, 15 sheep, and 24 seasonal hunts: https://fangrush.com
Studio: Fangrush Studio. We are looking to reach global web players and publish through Poki's platform.
```

## 提交前

- [ ] Demo可访问，游戏名与页面一致。
- [ ] 姓名、邮箱、国家和团队情况真实。
- [ ] 已确认当前封面、截图和游戏文案。
- [ ] 未提交密码、API Key、证件或支付凭据到仓库。
- [ ] 用户本人在官网点击提交。

获准后才启动真实 SDK、平台包、Inspector 和广告接入。通过准入不等于游戏已经上架。
